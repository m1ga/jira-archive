Description

I have added eventListener for orientation change Ti.Gesture.addEventListener('orientationchange', function(e) { if (e.orientation === Titanium.UI.PORTRAIT || e.orientation === Titanium.UI.UPSIDE_PORTRAIT) { alert(e.orientation); } else if (e.orientation === Titanium.UI.LANDSCAPE_LEFT || e.orientation === Titanium.UI.LANDSCAPE_RIGHT) { alert(e.orientation); } }); And when I change the orientation it is fire but when I close the window and open it again then it does not fire. I also remove the listener on window close event Ti.Gesture.removeEventListener('orientationchange'); if orientation is applied to one window on orientationchange event then orientationchange event for other window does not fire
