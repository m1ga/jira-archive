Description

It is an alloy project, currently just targeting ios, with a number of modules, including 3 which we have authoured to integrate to Healthkit, external accessories, and Bluetooth LE. Because we are using bluetooth it is necessary to debug on the device as well as use the simulator. Also we have been running the app in xcode to debug the native modules. The project can compile and run on the simulator, but now fails to compile where deploying to a device from appcelerator.
