Description

I have dashboard screen with many icons with four tabs, clicking the icons will open the seperate windows. Among the icons only one window is set to landscape and others are set in both landscape and portrait mode. When device is in landscape mode and launching the window(which is set to landscape forcefully), the screen momentarily appears as shrinked in landscape mode and then quickly stretches back to appear normally in landscape mode. Note : Am not using any setTimeout function before opening the window
