Description

When I first created an Arrow app using the Arrow CLI, it automatically created two instances of ArrowDB - one for dev and another for production. These ArrowDB instances were accessible as "Apps" inside the Arrow App under the "Apps" section in the Arrow Dashboard. However, recent upgrade (or something) has caused these apps to disappear. Now, those ArrowDB instances are no longer available in the dashboard ANYWHERE. Attached are screenshots to help explain the problem.
