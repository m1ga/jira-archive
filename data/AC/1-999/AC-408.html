Description

Steps to Reproduce

1. Run a mobile project (adroid emulator o device) 2. terminate and relaunch

Actual Result

On android emulator: new android emulator is started and app is never installed on new emulator instance. On device app is never relaunched

Expected Result

On emulator, app is relaunched on same emulator instance. On device, app is relaunched.
