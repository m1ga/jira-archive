Description

When doing a dev build using the CLI, iOS does not minify unless requested while android runs minify by default.

Comments

  1. Chris Barber 2014-03-17 Starting in Titanium SDK 3.2.0, both iOS and Android do NOT minify JavaScript for simulator/emulator builds. However, both iOS and Android DO minify JavaScript for device and distribution builds. You can disable JavaScript minification for device and distribution builds by using the --skip-js-minify flag. The --skip-js-minify flag has no effect for simulator/emulator builds. I am planning on implementing build configurations as apart of TIMOB-12025. The heart of that task is to decouple deploy type from things like JavaScript minification. You'll be able to control several knobs for the various deploy type/build configurations.
