  1. Mauro Parra-Miranda 2014-08-04 Hello! Likely your nvm is not showing a global nodejs, that's the reason your Ti Studio doesn't see it. Please read this article: https://www.digitalocean.com/community/tutorials/how-to-install-node-js-with-nvm-node-version-manager-on-a-vps and use the workaround to have a copy of nodejs in the usr local directory. Best Regards.
  2. Mauro Parra-Miranda 2014-08-07 Hello! Another workaround: https://coderwall.com/p/of5u8w (basically, install nvm from the Ti Studio console, so you are using exactly the same environment). Best Regards!
