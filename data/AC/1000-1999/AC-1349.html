Description

Use the latest KitchenSink, open Phone --> Geolocation With Xcode 6, in the iOS8 simulator the geolocation callback is not fired (see the simulator screen in KS). In the iOS7.1 simulator the callback is fired, usually with an error. EDIT: tested on iPhone 4S with 7.1.2, project built with Xcode6/SDK 8.0, and there too the geolocation callback is called with an error. With Xcode 5/SDK 7.1 no issues. Note that with the iOS7.1 SDK (Xcode5) it works fine every time on the simulator.
