Description

According to the documentation, when using a WebView any js file which is loaded locally but dynamically must have an extension of .jslocal to avoid it being removed at build time. However, when working with these files Studio does not recognise them as a valid javascript extension so none of the coding helpers are available. The editing experience becomes only a fraction better than notepad.
