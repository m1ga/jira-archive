  3. Ritu Agrawal 2014-04-17 Removing an item from a ListView section marks it ready for garbage collection but it does not mean that the object is collected immediately. Garbage collection timing is based on a number of different factors. A better test would be to add and remove items in a large loop and see if the memory usage continues to go up or stabilizes at some point.
