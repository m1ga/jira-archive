  1. Ritu Agrawal 2014-04-17 You should be able to resolve this issue by deleting the secure storage from Studio. Preferences -> General -> Security -> Secure Storage. In the Contents tab, select the '[Default Secure Storage]' item in the list and delete it. Restart Studio. Since the secure storage contents are deleted, Studio will prompt for user and password during startup. Restart again.
