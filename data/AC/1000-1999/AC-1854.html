Description

When doing a clean build, an error is encountered at line 627 of TiApp.mm for iOS 5.1 This works fine for iOS6 however. Issue can be fixed for iOS 5.1 by changing - (NSUInteger)application:(UIApplication *).... to - (NSObject*)application:(UIApplication *).... on line 625 Once changed, it doesn't compile for iOS6. So one has to switch when switching between iOS versions.
