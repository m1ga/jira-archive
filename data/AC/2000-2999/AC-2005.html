Daniel Sefton 2013-05-22 Hi Arvind, For these type of questions, please use the Q&A first, JIRA is for bug reports and feature requests: http://developer.appcelerator.com/questions/newest Anyway, can you go to "Titanium Studio -> Preferences... -> Studio -> Platforms -> iOS" and check that iOS SDK Home points to: 
    /Applications/Xcode.app/Contents/Developer/Platforms/iPhoneOS.platform/Developer/SDKs
     If not, please run this in Terminal: 
    xcode-select --print-path
     Check that it points to: 
    /Applications/Xcode.app/Contents/Developer
     If not run: 
    sudo xcode-select -switch /Applications/Xcode.app/Contents/Developer
