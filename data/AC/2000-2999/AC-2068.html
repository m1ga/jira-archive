Description

It seems like [www.codestrong.com](http://www.codestrong.com) is still not showing any dates for Codestrong 3.0, which is a serious blocker for releasing this awesome event. I tried both Safari and Chrome on Mac OS X and even on Android.

Steps to reproduce

1. Open your favorite browser 2. Visit [www.codestrong.com](http://www.codestrong.com) 3. Compare the result with the attached screenshot. It should say *Spring 2014*, but no concrete dates. 4. Hit the *Scroll down* button next to *Okay, you got my attention. Where do I find out more?* 5. Again, find no dates

Use cases

There are several use cases for having a date: 1. Plan a family holiday before or after the event 2. Allocate training budgets 3. Get the buzz going 4. Buy cheap tickets I'm sure there are more use cases. Feel free to add them as comments for this ticket.

Possible solutions

A. Pick a date (I'm sure there's an app for that) B. Cancel it (not recommended given [~jhaynie]'s recent [article on Re/code](http://recode.net/2014/02/13/why-satya-nadella-has-a-fighting-chance-of-saving-microsoft-from-itself/)) I hope to see this resolved as *Fixed* soon. This *IS* your bug ;)
