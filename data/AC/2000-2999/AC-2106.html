Description

When touching a row item, the behavior SHOULD be for the row background color to darken and for the ALL text in the row to turn WHITE. In Android, Background selected color for row is working with default row titles. But, when we have customized row , touching the label does not show the row darkening behaviour. Touching only on the other space like Left icon, Right disclosure is darkening the row. There is no option to highlight the row item text even on selection. In MobileWeb, there is no option to highlight the row item text on selection if we are using labels.
