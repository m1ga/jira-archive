Description

Steps to Reproduce

In a template, attach an event listener to one of the sub-views. Add 'touchstart' event listener. Trying to get the data of the row like this: var index - e.rowIndex; var section = e.section; var data = section.getItemAt(index). Will crash. Furthermore, e.rowIndex is undefined. e.itemIndex does return the right index for the row. When the event is switched to click instead, everything works.

Actual Result

Crash.

Expected Result

Getting the row data.
