Description

When attempting to launch the latest version of the RC of Titanium Studio (3.1.2.201307311059 and even the prior version), it freezes at the splash screen. I first experienced this behavior on an Ubuntu 13.04 64-bit installation, so I thought I'd try it on Windows 32-bit and I am getting the same behavior. I have tried this with both JDK 1.6 and 1.7 with the same results. JAVA_HOME is correctly set in the environment variables. I have also tried 3.1.1 and it works fine in the exact same environment. If I start with 3.1.1, have it working with no issues and then upgrade to 3.1.2 RC, upon restarting, I get the splash screen freezing behavior. So, installing 3.1.2 or upgrading from 3.1.1 both give this behavior on fresh VM's of Windows 7 x86 and Ubuntu 13.04 x64.
