Description

Steps to Reproduce

Install Studio, download updates, install SDKs, install dependences (including Node.js). Create template application using either 'Default Alloy Project' or 'Two-tabbed Alloy Application' and neither can be created due to a path problem with the template assets. Same problem for two-tabbed template.

Actual Result

'/usr/local/lib/node_module/Default Alloy Project' does not exist.

Expected Result

/usr/local/lib/node_module/default does.
