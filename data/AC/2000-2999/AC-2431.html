  8. Jerod Fritz 2012-11-08 The toImage() call on android was causing consistent crashing but this seems to have done the trick: 
             var imageFile = event.media;     
         var imageAsTaken = Ti.UI.createImageView({
           image : imageFile,
           autorotate : true
         });
         if (!ui.isAndroid()) {
           imageFile = imageAsTaken.toImage();
         } else {
           imageFile = imageAsTaken.toBlob();
         }
