Description

I'm using functions like Ti.UI.iPhone.createNavigationGroup(); in a commonJS module. An I'm using that module by var Window = require('/modules/Window'); It works with the simulator, but if I *publish the .ipa for distribution(Ad hoc)* and run on the device, this kind of error occurs : {{[ERROR] Application received error: 'undefined' is not a function (evaluating 'Ti.UI.iPhone.createNavigationGroup({ window: instance })') at Window.js (line 34)}} I found that if I use the Ti.UI.iPhone.createNavigationGroup() function once in app.js, than the problem is solved. It seems like functions only used in the modules are not included for build. Thank you.
