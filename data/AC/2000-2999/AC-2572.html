  2. Seth Toda 2014-01-20 Hello Jörgen, I took the project that you uploaded in a zip and was not able to reproduce your errors on the cloud. Navigating to the URL that studio outputs provides a functional JSON feed. I was able to access the data from a mobile app with a Ti.Network.HTTPClient. Here is what that code looked like: 
       var xhr = Ti.Network.createHTTPClient();
   xhr.onload = function(){
       var json = JSON.parse('['+this.responseText+']');
       
       //example access of length
       alert(json.length);
   }
   xhr.open('GET', 'url to your standard node app');
   xhr.send();
    More info on the HTTPClient can be found here... http://docs.appcelerator.com/titanium/latest/#!/api/Titanium.Network.HTTPClient Hopefully this sheds some light on the issue. - Seth
