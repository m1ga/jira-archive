Description

I'm running the iPhone SDK 1.8.x(Appcelerator Titanium) and for some unknown reason the value for CFBundleShortVersionString always remains in 1.0 and this issue was stopping me to update my app, then I found this workaround which solve my problem. When you build your app inside the build->iphone directory copy the generated info.plist file to the root project directory(where Tiapp.xml resides). Edit the info.plist that you just copied into the root directory, simply manually change the value for CFBundleShortVersionString(to whatever version you need). Save the file (the one in the root directory) and rebuild your project, the auto-generated info.plist will take the data in your amended file. Thank you
