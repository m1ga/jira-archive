Description

On iOS the filepath for local asserts is relative to / but on Android the file:///android_asset/Resource prefix is required. Note that file:/// alone does not work. If there is no fix then please document. Is suspect the default baseurl needs to be set for android 
    var win = Ti.UI.createWindow({title: 'Title'});
    
    var webview = Ti.UI.createWebView({
        borderRadius : 4,
        width : '100%', height : '100%',
        top : 0,
        backgroundColor:'white',
        scalesPageToFit:true,
        touchEnabled: true,
        });

    
    // Webview don't know about retina displays so fix with conditional
    var img_path = (Ti.Platform.displayCaps.dpi <300) ? '/img2x.png' : '/img.png';
    if(_platform==='android') img_path = 'file:///android_asset/Resources' + img_path;
        
    webview.html = '';
    win.add(webview);
    win.open()
