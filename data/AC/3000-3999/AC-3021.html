Description

Launcher icons in 3 sizes that are put into the platform/android/res/drawable-DENSITY folders (e.g. drawable-mdpi, drawable-ldpi, drawable-hdpi) are never used as launcher icons when the app is installed on an Android device. Ti Studio forces you to specify a program icon in the tiapp.xml Overview page, and that icon file has to be in the Resources directory (you get an i/o error 2 if there is no icon in the Resources dir). Titanium apps are using the Resources copy of the icon for everything. Q&A entry here: http://developer.appcelerator.com/question/124701/launcher-icons---how-to-get-titanium-to-correctly-use-the-right-size
