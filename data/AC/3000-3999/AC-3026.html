Comments

  1. Mauro Parra-Miranda 2012-05-02 In general, we don't recommend to put a scrollable view inside other scrollable view. The recommendation is create a view that encloses the webview, like a tab. Then you should be able to scroll with the scrollview via the view and scroll the webview inside the view. Best, Mauro
  2. Mauro Parra-Miranda 2012-05-14 Unsupported behavior. Not put a scrollable event inside an scrollview.
