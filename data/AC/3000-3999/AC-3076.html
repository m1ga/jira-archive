Description

Steps to Reproduce

During the initial setup and configuration of Titanium Studio, I attempted to configure the Android SDK support following the steps in the Getting Started documention. 1. From the Titanium Studio Dashboard, select the Configure tab 2. Click the Android button (the large green button) 3. Click "Install or Update Android SDK"

Actual Result

A dialog appears, noting that the path is empty, citing the root cause as an NLS Missing Message: MobileSDKConfigurationProcessor error. The dialog provides no clue as to the expected path and it's is not clear how to remidiate the issue manually, if that's possible.

Expected Result

Install of the Android SDK components required to develop native Android apps.
