Description

I have a webview that fire events inside the webview when user clicks on links and interacts with form elements. This functionality works fine when building for emulator or device but when building for distribution nothing happens when clicking links or any other form elements in the webview. Maybe the distribution target removes some important files, could be related to TIMOB-3395. As a workaround I use the build for emulator and signs it with my store key, but I don't know if there are any other things I miss when doing it this way. This should definitely be fixed before 1.7 final release.
