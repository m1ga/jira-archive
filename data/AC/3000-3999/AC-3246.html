  1. Amimul Hossain 2014-10-14 Hello, I have tested this issue and it's working fine. I get it to update to the coordinates within 10 - 15 seconds.

    TESTING ENVIRONMENT

    MacOS X 10.9.5, iPod iOS 8.0 Mobile SDK: 3.4.0.GA

    CODE SAMPLE

    tiapp.xml

    app.js

       if (Ti.Geolocation.locationServicesEnabled) {
       Ti.Geolocation.purpose = 'Get Current Location';
       Ti.Geolocation.accuracy = Ti.Geolocation.ACCURACY_BEST;
       Ti.Geolocation.distanceFilter = 10;
       Ti.Geolocation.preferredProvider = Ti.Geolocation.PROVIDER_GPS;
   
       Ti.Geolocation.addEventListener('location', function(e) {
           if (e.error) {
               alert('Error: ' + e.error);
           } else {
               alert("Lat: " + e.coords.latitude + ";\nlong: "+ e.coords.longitude);
           }
       });
   } else {
       alert('Please enable location services');
   }
