Description

It is very difficult as developers to know which features are supported by each platform when building Android / iOS apps, and I'm sure this problem will be compounded when Blackberry support is released. I really think a matrix of supported features by API call would make life a lot easier for everyone, something like http://www.phonegap.com/about/features/ would be great. Failing that, at least a little indicator in the documentation which shows which platforms are supported would be useful.
