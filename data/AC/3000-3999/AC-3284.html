Description

I am using Titanium 3.2 which has many modules etc that supersede previous versions and deprecate them. When I look at the modules in the Marketplace I don't see any way to search for 3.2x unless there are none? Currently the options look like this on the left hand side for all categories with no additional filters: Minimum Titanium Version Titanium 1.6 Titanium 1.7 Titanium 1.8 Titanium 2.0 Titanium 1.5 Titanium 2.1 Titanium 3.0 Titanium 3.0.2 Titanium 3.1 Titanium 3.1.2

Comments

  1. Ritu Agrawal 2014-03-21 Titanium 3.2 and 3.2.1 are available in the "Minimum Titanium Version" category. Note that it represents the minimum Titanium SDK version required for a module so most of the modules that automatically work with 3.2 release would not show up with this filter as they work with previous versions as well.
