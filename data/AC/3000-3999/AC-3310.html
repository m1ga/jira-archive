Comments

  1. Amimul Hossain 2015-08-31 Hello, If you are trying to run your project in iOS simulator 7.1 with xcode6, You need have iOS simulator 7.1 downloaded on that xcode6 version. Please Go to your xcode6/preference/downloads/iOS simulator 7.1 in component section. After downloading the simulator version you will be able to run project on simulator 7.1 versions. Also, Use Appcelerator studio as Titanium is obsolete. Follow the instruction [here](https://web.appcelerator.com/product/studio) to download the latest Appcelerator studio. Also, follow the instruction [here](https://web.appcelerator.com/product/cli) to install CLI, SDK. Hope this helps. Thanks.
