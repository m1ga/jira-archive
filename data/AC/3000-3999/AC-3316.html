SSL /TLS network communication supported or not with Certificate authority and without Certificate authority. - Yes, there is SSL support/ 3. Runtime Exception handling - Yes, you can catch issues and act accordingly. 4. Local data storage, session, cookies,cache etc - Yes, you have some of this. But the apps ARE NOT HTML5. Backup / Restore: In case if page crashes than can we store in backup file in devices? - What do you mean with page crashes? 7. Device storage for IOS - Yes, you can store data either in files or sqlite databases. For more info: http://docs.appcelerator.com/titanium/latest/#!/guide Best regards
