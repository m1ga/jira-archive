  4. Michael Kellogg 2016-03-25 I have resolved this issue. For future reference for you guys I will tell you what fixed it for me. I had been using gittio in order to download and connect modules up to my projects. I'm not sure which one had broken appcelerator for me, but once I removed these modules from my Library -> Application Support -> Titanium -> Modules folder appcelerator was able to execute projects again normally. If you see issues like this in the future maybe check if the user has recently installed modules. Thanks for your support guys.
