Description

Problem: To build for windows the setup checks for the presence of the emulator. If the emulator is not installed it throws an error: An error was detected with the SDKs configuration. Reason [Windows SDK Home] Windows platform is not supported by the selected SDK". Reason: Not everyone needs the emulators installed. Solution: Only check for the presence of the SDK.
