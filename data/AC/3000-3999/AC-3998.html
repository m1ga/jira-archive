Comments

  1. Aminul Islam 2016-07-11 Hi, Thanks for reaching us. So far, we have been unable to reproduce it. From the Studio menu bar, select Help > Install New Software... Click the Add button, to add the Google ADT Plugin update site https://dl-ssl.google.com/android/eclipse/ Wait for the package list to be populated Select all the resulting ADT packages Click the Next button Click the Next button on the Install Details screen that follows I recommend you upgrade Appcelerator Studio to the latest build and let us know if you get the error again.
