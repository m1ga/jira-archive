I have both Xcode-beta.app and Xcode.app installed. The Xcode version is set to Xcode.app (see screenshot attached). Actual Result

  4. Chris Barber 2016-09-11 This is expected behavior. The Xcode path in the Studio preferences has no effect. The selected iOS version is what dictates which Xcode to use. If you select iOS SDK 10.0, then Titanium's build system will find the most suitable version of Xcode that supports iOS SDK 10.0. In this case, the iOS SDK 10.0 is selected and thus it chooses Xcode 8.
