Description

Steps to Reproduce

Using Appcelerator Studio, build: 5.0.0.201712081732 Installed Oracle JDK 1.8.152 ( jdk-8u151-macosx-x64.dmg ) from this URL: http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html *I already had / have JDK 1.8.131 installed.

Actual Result

Appcelerator opens but I am unable to select anything from menu or click any open tabs. I had to force quit a few times and tried restarting mac osx 10.13 with same results. I found and used JPS tool to identify eclipse issue from URL http://docs.appcelerator.com/platform/latest/#!/guide/Crashes_and_Freezes I have attached jps log file. After removing JDK 1.8.152 appcelerator is now working as expected.

Expected Result

appcelerator studio UI is should work as expected or display an error.
