  1. Sharif AbuDarda 2018-05-26 Hello, You can download the Android SDK package in here http://dl.google.com/android/android-sdk_r21.1-macosx.zip. Now, put the unzip folder in a root directory and configure the directory in your studio by studio/preference/studio/platforms/android/ Android SDK home. Close studio. Go to your Android SDK folder/tools/android. This will open Android SDK manager. Download and install the recommended SDK tools and build tools. Now restart the pc after the installation is done. Now check again in the studio. Android SDK should be configured. If it does, Go to your Android SDK folder/tools/android. This will open Android SDK manager. Download and install the recommended API, SDK tools and build tools 23,24,25.... Now restart the pc after the installation is done. Android SDK should be configured. If so, all is good. Let us know with "appc ti setup check"log. Thanks.
