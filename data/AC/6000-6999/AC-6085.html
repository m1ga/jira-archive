  2. Matthew Delmarter 2019-01-03 Actually after creating a clean test case I managed to get this working. For others who look here, here is the cleanest working sample code, using [this](http://abload.de/img/dottjz0w.png) 1 x 1px transparent png image: 
       var win = Titanium.UI.createWindow({
       title: "Magic!",
       largeTitleEnabled: true,
       barImage: "dot.png",
       hideShadow: true
   });
   
   var nav = Ti.UI.iOS.createNavigationWindow({
       window: win
   });
   
   nav.open();
    This code works with largeTitles and normal window titles. This issue can be closed. So sorry for mucking you around.
