Comments

  1. Ewan Harris 2020-05-19 [~jignesh.igp], thanks for the report. We're aware of this issue and it will be fixed in 9.0.2 (see TIMOB-27852) which should be released in the next week or so. But you can get the fix today by running \[appc\] ti sdk install -b 9_0_X and setting the version installed as your projects SDK.
  2. Ewan Harris 2020-05-19 Closing as a dupe of TIMOB-27852
  3. Joshua Quick 2020-05-19 [~jignesh.igp], Titanium 9.0.1 is building the AAB file. It's just not copying it to your chosen distribution directory. You can find the AAB file under the following location. 
       ./build/android/app/build/outputs/bundle/release/aap-release.aab
    Like Ewan mentioned, this issue will be resolved in 9.0.2.
