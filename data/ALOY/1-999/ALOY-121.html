Description

Example: 
"backgroundGradient": {
    "type": "linear",
    "startPoint": { "x": "0%", "y":"0%"},
    "endPoint":   { "x": "0%", "y":"100%"},
    "colors": [
        { "color": "#000", "offset": 0.0 },
        { "color": "#666", "offset": 1.0 }
    ]
}
generates this incorrect output: 
backgroundGradient: {
    type:"linear",
    startPoint: {x:"0%",y:"0%",},
    endPoint: {x:"0%",y:"100%",},
    colors: [
        0: {color:"#000",offset:0,},
        1: {color:"#666",offset:1,},
    ],
}
