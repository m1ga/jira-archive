Description

We need much more helpful command line help. We list the valid commands and options, but we don't list them in any detail whatsoever, nor do we have examples. In addition, we should probably consider a syntax where you can do a generic alloy help call, or a help call that is specific to each command, like this: alloy help compile. The command-specific help could function just like it does for git.
