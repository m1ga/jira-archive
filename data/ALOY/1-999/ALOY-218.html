Description

Widget creation from the CLI is very limited now. You can only specify the name and description now. We should expand it in a way that more of the basic widget.json values can be set during generation. This may not be as useful at the CLI, but it will make any TiStudio integration with the CLI much more powerful. Think of it much like the native module generation interface in TiStudio where author, version, and lots of other info can be filled in. We need to investigate a solid way to expose at least a majority of the widget.json fields via CLI to facilitate the above.
