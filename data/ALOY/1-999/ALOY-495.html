Description

problem

If you put the following style in a TSS, it would match both Ti.UI.View and Ti.Map.View elements, which it should not: 
"View": {
    height: 60
}

the fix

The logic for checking what styles to apply only takes into account the nodename, not the full namespace name. We need to use the full namespace for these style assignments. The unique case of Ti.Map.View should be listed by full namespace in the TSS files if it is used.
