Description

You should be able to use the or operand '|' in TSS files. There are some properties that are creation-only and accept this operands, like windowSoftInputMode on android. For example: 
var win = Ti.UI.createWindow({
    windowSoftInputMode : Ti.UI.Android.SOFT_INPUT_STATE_ALWAYS_VISIBLE | Ti.UI.Android.SOFT_INPUT_ADJUST_PAN
});
 Alloy will fail if you try to do that in a TSS file.
