Comments

  1. Tim Poulsen 2014-04-18 PR: https://github.com/appcelerator/alloy/pull/366 Functional test: As of this PR, Studio & CLI still include Tizen support. So, create a Titanium project that includes Tizen as a target platform. Run alloy compile for your project and you should receive a compile error stating that Tizen is not a supported platform. Remove Tizen from the list of supported platforms for your project, and you should be able to compile without errors.
