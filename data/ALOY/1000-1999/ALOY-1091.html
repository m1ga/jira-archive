Description

If you use id name except 'picker' to Picker, that id will be handled as variable.

view

style

'#testpicker': {
  type: Ti.UI.PICKER_TYPE_DATE_AND_TIME
}

compiled code

$.__views.testpicker = Ti.UI.createPicker({
  type: Ti.UI.PICKER_TYPE_DATE_AND_TIME,
  id: testpicker
});
