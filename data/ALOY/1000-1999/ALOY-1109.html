Description

The *.tss is basically a JavaScript Object without a leading \{ and a terminating \}. If it was allowed to also write the style within curly braces the code completion would only need to be set to JavaScript. I.e.: \{ "Label": \{ width: Ti.UI.SIZE, height: Ti.UI.SIZE, color: "#000" \} \} I would love to see this as a feature
