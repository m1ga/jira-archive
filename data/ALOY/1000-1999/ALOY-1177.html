Description

When declaring a SearchView element in XML markup, Alloy should assign it the 'Ti.UI.Android' namespace. Alloy is currently doing this for a lot of iOS-specific components. These two items should be equivalent. 
<SearchView ns="Titanium.UI.Android"/>

<SearchView/>

Comments
