Description

Present behaviour

 var imageView = $.UI.create("ImageView",{
     apiName: "ImageView",
     classes: ["profile-img"],
     id: "profileImg"
});
imageView.image = "url";
Image url will be dynamic, so good to make a way to pass it at creation.

Expected behaviour

 var imageView = $.UI.create("ImageView",{
     apiName: "ImageView",
     classes: ["profile-img"],
     id: "profileImg",
     properties: {
      image: "url"
   }
});
