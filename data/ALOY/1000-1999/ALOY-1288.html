Description

The [default app.tss](https://github.com/appcelerator/alloy/blob/master/Alloy/template/app.tss) sets the Window backgroundColor to white for all platforms. on Windows the default color of a label is white, making them invisible when using the default app.tss. Just like we do for Android, we should set the Label color to black. Note: There might still be issues with other elements that default to white on Windows, like titles in ListViews.
