Description

I'd like to be able to do: 
<Alloy>
  <Label text="L('key') :" />
</Alloy>
Currently this compiles to: 
    $.__views.__alloyId0 = Ti.UI.createLabel({
        text: 'L("time") :',
        id: "__alloyId0"
    });
While I'd like it to be: 
    $.__views.__alloyId0 = Ti.UI.createLabel({
        text: L("time") + ' :',
        id: "__alloyId0"
    });
