  1. Ewan Harris 2017-11-06 Default node install path is %ProgramFiles%\\nodejs not %ProgramFiles%\\Node.js. However I propose that we switch this to query the registry on Windows for the install path as the installer allows for the path to be changed. Would probably need to query HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Node.js and HKEY_LOCAL_MACHINE\SOFTWARE\\Node.js for the InstallPath value
