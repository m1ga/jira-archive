Description

json strings are coming in single quotes but it should be in double quotes. *To reproduce*

Run appc appcd exec /system-info/1.x/info --json --no-banner

Valdiate the output here https://jsonformatter.curiousconcept.com

Studio json parser validation is failing due to this.

Comments

  1. Chris Barber 2017-11-16 Excellent find. Note that appcd exec will ALWAYS return JSON unless the exit code is non-zero.
  2. Chris Barber 2017-11-16 Also note that the \-\-json and \-\-no\-banner have no effect as far as appcd is concerned. Pass them in if you need to keep Appc CLI happy.
