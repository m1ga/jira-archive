Description

Alloy must be a first class citizen. No more Alloy CLI. Every new app should be an Alloy app by default. This means bringing the Alloy project template files and the Alloy compiler logic into the Titanium SDK. This ticket includes: * Integrating Alloy project creation and building into the Appc Daemon Titanium plugin * Enable Alloy build to take advantage of the Titanium build pipeline including the build state and differential build system This ticket does NOT include: * Changing Alloy's API, features, or design: refer to DAEMON-76 for Alloy 2
