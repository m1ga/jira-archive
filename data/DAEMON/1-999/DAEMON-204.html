Description

From my research into windowslib v2 we're going to need to query the 32 bit registry hive architecture, this can be achieved by passing an opt.arch value to win-reg. If no value is specified we should default to null. This will also need to be added to appcd-detect

Comments

  1. Chris Barber 2019-05-06 We are abandoning the spawning of the reg.exe and thus we no longer need to be concerned with this ticket. Instead of spawning reg.exe, a new library called winreglib is written in C and uses the win32 API to directly talk to the registry (DAEMON-81).
