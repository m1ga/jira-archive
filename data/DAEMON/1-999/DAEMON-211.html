  3. Satyam Sekhri 2017-12-15 The issue seems to be with Daemon. Steps: 1. Remove all the provisioning profiles from "~/Library/MobileDevice/Provisioning Profiles" 2. From terminal run the command "appc appcd /ios/1.x/info". It still returns all the deleted profiles 3. Create and add a new provisioning profile 4. Again run the command "appc appcd /ios/1.x/info". It does not show the new provisioning profile and still returns all the deleted ones. 5. Restart daemon 6. Again run the command "appc appcd /ios/1.x/info". Now it shows only one provisioning profile that was added in step 3.
