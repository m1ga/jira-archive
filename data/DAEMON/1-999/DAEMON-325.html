Description

The "ti new" command (and "ti add") need to create projects and components based on a template. A general purpose template toolkit would standardize template structure, make it easy to add templates to other Node.js-based products, and get more people battle testing the library. The template library specification can be found here: https://techweb.axway.com/confluence/display/pe/Proposal%3A+Project+Template+System.
