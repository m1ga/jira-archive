Description

When getting the status of a plugin where the version is a semver range, it will scan all plugins for a match by name and version. The order of the registered plugins is whichever order they were loaded. The list of plugins must be sorted in order to deterministically find the plugin. If you have @appcd/plugin-titanium v1.6.0 and v1.8.1 installed, getting the status for ^1.6.0 should return v1.8.1. 
appcd exec /appcd/plugin/status/@appcd/plugin-titanium/^1.6.0
