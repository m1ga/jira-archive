Description

Node 8 is no longer in maintenance after Dec 31, 2019 and npm dependencies are starting to depend on Node 10 as a minimum. In Daemon 4.0, we need to bump the minimum supported Node version to >=10.19.0. Originally we choose 10.2.0 so that we get N-API 3 needed by native deep dependencies. Then we bumped to 10.13.0 since that was the first LTS release. Now we have bumped it to 10.19.0 because got (the HTTP request library) relies on the new DNS APIs.
