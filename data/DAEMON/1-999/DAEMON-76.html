Description

With Alloy apart of the Titanium build pipeline, we can introduce a new Alloy compiler that allows us to redesign Alloy to be modern and competitive. Things to consider are: * Components-based (like React, Angular, etc) * ES2015-based design (classes, decorators) * Titanium API changes ** Improved box model ** CSS support (flexbox, etc) - likely a subset * i18n support * State stores instead of models - i.e. redux, mobx, vuex * Hot reloading * ES6 template strings or JSX or Dust.js or some other template engine? * ES6 module support * Universal asset catalog
