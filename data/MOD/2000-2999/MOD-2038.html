Description

The image dimension properties such as width and height do not affect the image. It works fine on iOS but not on Android. The example provided in this module was tested by setting different dimensions and it did not work.

Steps to Reproduce:

1. Load the example app.js and text.html. 2. In text.html modify the dimensions height and width in HTML/CSS for any of the Local or Remote Images. Expected behavior: The image should change accordingly. Actual Behavior: The image remains as it is.
