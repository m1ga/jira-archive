Description

For most modules on https://github.com/appcelerator-modules/ we have GitHub issues enabled and a fair number of them have old, open issues and PRs: * https://github.com/appcelerator-modules/ti.storekit/issues * https://github.com/appcelerator-modules/ti.styledlabel/issues * https://github.com/appcelerator-modules/ti.imagefactory/issues We should go through all of these modules and decide to either: * [Archive](https://github.com/appcelerator-archive) the module and clearly mark as unmaintained. ** Modules like ti.urbainairship and ti.googleanalytics that conflict our interests ** Modules like ti.styledlabel that have been made obsolete by core APIs (AttributedStrings) ** Modules in bad shape and no priority to fix that * Go through issues, close or move them to JIRA and then disable GitHub issues /cc [~hansknoechel]
