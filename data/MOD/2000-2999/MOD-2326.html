Description

UDP Socket communication implementation even in the iOS and Android platforms. Currently my company uses the iOS ti.udp module that is no longer maintained. https://github.com/appcelerator-archive/ti.udp Given the growing IoT market that makes intensive use of the connectivity TCP and UDP, there is required the implementation of the UDP and TCP socket communication in the same way in the 3 platforms (iOS, Android, Windows Phone). Currently the documentation report that in the Windows Phone SDK the UDP socket management is present, it may be possible to incorporate the UDP iOS module (ti.udp) into the standard Titanium SDK, and complete the Android part.
