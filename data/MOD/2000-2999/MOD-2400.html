Comments

  1. Joshua Quick 2018-02-28 This feature should be added to our "ti.admob" module. The reason we don't want to add this to our core code is because Google Play scans for this API usage, and if they find it, then they'll demand the app developer to write-up and upload a "Privacy Policy" if ads are not being used. We don't want to penalize other Titanium app developers who don't use advertiser ID. So, putting it into the admob module avoids this issue.
