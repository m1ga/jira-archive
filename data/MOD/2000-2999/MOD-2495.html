Description

Android and iOS have a lot of parity issues now due to different implementation of interstitial and rewarded video ads. We should follow the direction where Android is going with different proxies for the different ad types to achieve parity between the two platforms. At the same time we can use this opportunity to move the module to a Swift code base.
