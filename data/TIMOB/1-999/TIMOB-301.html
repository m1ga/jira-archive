Description

The camera permission is only added if the video player API is used.

Apply the attached patch or update .../android/builder.py to include the following line in the permissions section.

'Media.showCamera' : CAMERA_PERMISSION,

Should look similar to.

# MEDIA
'Media.vibrate' : VIBRATE_PERMISSION,
'Media.createVideoPlayer' : CAMERA_PERMISSION,
'Media.showCamera' : CAMERA_PERMISSION,
