Description

See the following code:

var label = Ti.UI.createLabel({
    text:'some text',
    font:{fontSize:14},
    height:20,
    width:300,
    left:10
});
Ti.API.info("label.size.height = " + label.size.height);
Ti.API.info("label.size.width = " + label.size.width);

Output is:
[INFO] label.size.height = 300 [INFO] label.size.width = 300
