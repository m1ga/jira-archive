Description

It's possible for the launch process to show the whole set of "building, compiling, installing, launching, deployed" INFO messages even if they're not true (i.e., the emulator is not even finished launching, so nothing is really being installed.) This happened to me three times. All three times it was when the launch process had to create a new AVD (because of new combo of SDK / screen size.) So there might be a connection there.
