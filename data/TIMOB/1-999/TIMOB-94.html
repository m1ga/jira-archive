Description

Enable the ability to set the badge for any tab. The API is:


// get tabs array (or you can get by name)
var tabs = Titanium.UI.getTabs();

// set first tab to 1
tabs[0].setBadge(1);

// set second tab to 2
tabs[1].setBadge(2);

// set third tab to 3 
tabs[2].setBadge(3);

// set fourth tab to 4
tabs[3].setBadge(4);
