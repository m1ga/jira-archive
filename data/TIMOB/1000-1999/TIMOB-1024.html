Description

A 1.3.0 iPhone application does not run on the iPad in compatibility mode - the compatibility mode frame shows up, but the app does not load inside it. This can be recreated by creating a new iPhone project and attempting to run the project using the 3.2 SDK. Confirmed by me and a number of other community members.

Titanium Mobile 1.2.1 iPhone projects do not have this issue - they run in compatibility mode on the iPad as you would expect.
