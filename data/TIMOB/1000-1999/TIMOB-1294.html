  1. Junaid Younus 2012-08-14 Not sure if I understand the issue here correctly, but here is some sample code that I came up with: 
       var win = Ti.UI.createWindow({
       backgroundColor : 'white', 
       orientationModes: [Ti.UI.PORTRAIT, Ti.UI.LANDSCAPE_LEFT, Ti.UI.LANDSCAPE_RIGHT]
   });
   
   var label = Ti.UI.createLabel({
       text: 'Null'
   });
   
   var button = Ti.UI.createButton({
       title: 'Query orientation',
       top: 20,
       width: 150,
       height: 70
   });
   
   button.addEventListener('click', function(e)
   {
       label.text = Ti.Gesture.orientation;
   });
   
   win.add(button);
   win.add(label);
   win.open();
    Tested on a Samsung Galaxy S2 using TiSDK 2.2.0v20120814103312, seems to work as documented. Closing ticket.
