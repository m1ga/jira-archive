Description

When using a buttonbar with a set barColor (black in this case), any buttons with styles set to a system style (done, plain, bordered) render using the parents barColor.

Removing the barColor from the bar renders the buttons as thier normal blue/grey/etc.

This leaves me to use image assets to 'recreate' native looking elements when using any specified barColor.
