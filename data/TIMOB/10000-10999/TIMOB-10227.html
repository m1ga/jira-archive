Description

Kitchensink crashes on relaunch after opening a tab and leave the device idle for sometime. This is not regression.This issue occurs on 2.1.0 as well but issue does not reproduce on 2.0.1. Steps to reproduce: 1. Launch kitchensink. 2. Go to Kitchensink> Phone> sound> Local with file url. 3. Leave for some time. 4. Close the app using hardware back button. 5. Relaunch kitchensink. Actual: Kitchensink crashes. Expected: Application should not crash and should open normally
