Description

The .png files in the Resources/iphone directory appear twice in "ipa" file. Once in the root of the package and once in the iphone directory.

Steps to reproduce

1. Create a sample project in the studio. 2. Add some png files in the Resources/iphone directory. 3. Build the app for distribution (Distribute - Ad Hoc/Enterprise). 4. After the ipa file is generated, zip the ipa file 5. Now unzip the file and open the Payload folder. 6. Right click on the file in the Payload folder and select "Show Contents". 7. You can see the png files appear twice in the file as compared to the project in the studio.
