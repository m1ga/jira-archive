Description

Table views rows can be created using 2 ways: 1. Explicit - Ti.UI.createTableViewRow 2. Implicit - via JavaScript dictionary object. For example: 
 var data = [{title:'Row 1', hasChild:true, color:'red', selectedColor:'#fff', special:'special 1'},];
 var table = Ti.UI.createTableView({data: data});
The 2nd method isn't the recommended one and should be deprecated.
