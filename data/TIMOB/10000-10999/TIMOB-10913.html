Description

The hintText value is blocked by passwordMask in a textField. Screenshots attached. This is a regression. This did not occur in SDK 2.1.2.GA in Firefox. This could not be tested in SDK 2.1.2.GA in Chrome due to TIMOB-9717. Steps to Reproduce: 1. Run code: 
var win = Ti.UI.createWindow({
	backgroundColor : 'black'
});

var textField = Ti.UI.createTextField({
	backgroundColor : 'white',
	height : '10%',
	width : '50%',
	hintText : 'hintText',
	passwordMask : true,
	color : 'black'
});

win.add(textField);

win.open();
Actual Result: hintText is blocked by passwordMask Expected Result: hintText should be visible. When text is entered in textField, the hintText should disappear and the typed text should be blocked by passwordMask.
