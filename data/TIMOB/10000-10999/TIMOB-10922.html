Description

Need a "info" command to detect and display information about the development environment. This includes the following: * Operating System * Xcode/iOS * Android * Titanium SDK For the Android section, we need to know where Android is installed, what version (if even possible), what SDKs/targets are installed (including Google APIs), what AVDs have been created, what version of adb, and whatever other data makes sense. The "android list avd" and "android list target" commands should give you most of the data. Also need to find the location of the Android NDK.
