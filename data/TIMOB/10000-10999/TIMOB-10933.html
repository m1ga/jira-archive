Description

Pressing tab while holding the iphone in landscape mode opens the window in Landscape mode first and then after some time returns to portrait mode automatically. This is not a regression.This even occurs on 2.1.2.GA Steps To reproduce: 1. Run the Kitchen Sink application 2. Rotate the device in Landscape mode 3. Tap on Platform Tab.(Wait for few seconds) Actual Result: The window opens in lanscape mode for some time and then returns to portrait mode automatically. Expected Result: The window should remain in Lanscape mode.
