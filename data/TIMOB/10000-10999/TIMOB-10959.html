Description

The bbndk toolchain doesn't support paths containing whitespaces. On mac the build step happens within the path containing the mobile sdk, namely: "~/Library/Application Support/Titanium/mobilesdk/osx". Building from that folder fails because of the space in "Application Support" and using a symlink doesn't help. To workaround that, the idea is to modify the build scripts to build from a temporary folder, likely /tmp or it could be configurable by the user from the Studio preferences.
