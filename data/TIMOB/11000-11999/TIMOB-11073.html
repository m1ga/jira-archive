Description

This is not a regression. It occurs since 2.0.1. Steps to reproduce 1. Launch KitchenSink 2. Go to "Base UI"- "Views" - "Table Views" 3. Tap on "Touch Events" 4. Longpress each row(on the text) then without lifting finger,swipe the label(may need to try 3-4 times,start with last row) Expected Result: 4. After step 4, the label should only change to blue and back to white when finger is lifted off Actual Result: 4. After step 4, the label gets disappeared. It appears again on clicking back on the tableviewrow.
