Description

This is a regression. It works fine with 2.1.3GA. The different screen shots are attached with the bug.The apple logo and the html script text 'Click Me to Hide the Toolbar' are not appearing with 3.0.0 sdk. Seems html script is not being read . It appears with 2.1.3 sdk. Steps to reproduce: 1) Install the refactored kitchen sink on android device and run it. 2) Go to Base UI -> Views-> Web View -> Local URL Expected result: 1) The app should run successfully over device. 2) It should display the screen as shown for 2.1.3. And clicking the text 'Click Me to Hide the Toolbar' should hide the toolbar shown at the bottom. Actual result: 1) The app runs successfully over device. 2) It displays no apple logo and no text. Not able to hide toolbar.
