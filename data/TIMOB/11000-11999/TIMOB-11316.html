Description

This is a regression bug. Mobile web distribution folder size in 2.1.3 SDK is 4.6MB Steps To Reproduce: 1.Launch Titanium Studio. 2.Create > Titanium Project and select Deployment target as > Mobile Web. 3.Select 'Publish' that project as > 'package - Mobile web.' 4.Select option : Copy mobile web app to a local file system. and browse to any desired folder path. 5.Make sure you have selected SDK as 3.0.0.v20121008225713 6.Verify mobile web folder size . Actual: In 3.0 SDK build, folder size is 4.7MB whereas in 2.1.3 its size is 4.6MB
