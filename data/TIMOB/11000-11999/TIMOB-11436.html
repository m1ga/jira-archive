  1. Clayton Headley 2012-10-17 Tested in Titanium Studio, build: 3.0.0.201210151149 using Titanium CLI v3.0.6. The "titanium info" command does not correctly store android information (see screenshot). Reopening ticket.
  2. Chris Barber 2012-10-17 The "titanium info" command does not "store" anything. It simply queries the system. The Titanium CLI does NOT use the Android SDK path from Titanium Studio's preferences. To set the Android SDK path for the CLI, you must run "titanium config android.sdkPath /path/to/android-sdk". Setting this ticket back to fixed.
