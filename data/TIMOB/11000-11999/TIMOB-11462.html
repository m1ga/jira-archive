  1. Bill Dawson 2012-10-15 It speaks the default text fine, so this is actually a duplicate of TIMOB-11463. Regarding the default text: clicking a table row is not the same as the table row having focus. Accessibility text is read aloud by TalkBack when a view has focus, not when the view is "clicked". So to test on Honeycomb and below, you need either... a) A device with a D-PAD or equivalent. b) The "Eyes Free Keyboard" to emulate a D-PAD on the screen. Eyes free: https://play.google.com/store/apps/details?id=com.googlecode.eyesfree.inputmethod.latin&feature=search_result#?t=W251bGwsMSwxLDEsImNvbS5nb29nbGVjb2RlLmV5ZXNmcmVlLmlucHV0bWV0aG9kLmxhdGluIl0.
