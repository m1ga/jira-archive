Description

Problem description

When a button in Android is set to "visible: false", its border keeps being visible.

Steps to reproduce

Use this code to see the issue: 
var win1 = Titanium.UI.createWindow({  
    backgroundColor:"#000"
});

var button1 = Ti.UI.createButton({
    title: "Invisible",
    borderWidth:1,
    visible:false,
    borderColor:"white"
});

win1.add(button1);

win1.open();
