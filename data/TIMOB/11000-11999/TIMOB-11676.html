Description

Description: While testing Kitchen sink, the search bar does not resize when rotating the device to landscape then back to portrait. Not a regression, did occur in 2.1.3 Steps to reproduce: 1) Install KS 2) Launch KS > Base UI > Views > Table Views > Headers with Filter 3) Rotate the device to landscape Searchbar autofill size of the screen. 3) Rotate the device back to portrait Actual result: Searchbar is shorter then the width of the screen. See screenshots. Expected: Searchbar resizes based on orientation.
