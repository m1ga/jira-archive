Description

Setting *backgroundImage* and *backgroundColor* in the same time to the window will leave only the last in the code: This will show only the image: 
var win = Ti.UI.createWindow({
	backgroundColor:'#f00',
	backgroundImage:'/sun.png'
});

win.open();
This will show only the red background: 
var win = Ti.UI.createWindow({
	backgroundImage:'/sun.png',
	backgroundColor:'#f00'
});

win.open();
Exactly the same for Ti.UI.View.

  2. Pedro Enrique 2013-08-09 Hi Dan, In BlackBerry Cascades, the UI framework, there is no backgroundImage or backgroundColor property on UI components. There is only a "background" property, and this takes either a color or an image. Unlike iOS, here you can only specify one or the other. http://developer.blackberry.com/cascades/reference/bb__cascades__container.html#property-background
