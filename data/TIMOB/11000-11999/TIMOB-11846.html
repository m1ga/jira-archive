Description

The layout for windows is not working properly. It seems to be defaulting to "stack" layout. It should be using absolute layout by default to layout its children.

Test Case

var win = Ti.UI.createWindow({
    backgroundColor: '#fff'    
});

var startSocket = Ti.UI.createButton({
    title: 'Trim Drone',
    top: 50, left: 10
});
win.add(startSocket);

var sendTo = Ti.UI.createTextField({
    value: '192.168.1.1',
    top: 300, left: 10
});
win.add(sendTo);

win.open();
