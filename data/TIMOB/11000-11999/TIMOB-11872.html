Description

Nimbus kit is a great framework for ios development You can find it [here](http://nimbuskit.info/) By using it we could get feature very close to android, and would gain much time implementing new features. For example html property for label http://latest.docs.nimbuskit.info/NimbusAttributedLabel.html or photoAlbum like gallery on android http://latest.docs.nimbuskit.info/NimbusPhotos.html Improved webview with toolbar http://latest.docs.nimbuskit.info/NimbusWebController.html We could also use to better manage network images http://latest.docs.nimbuskit.info/NimbusNetworkImage.html NiAttributeLabel already handle textalign and verticalAlign And we could even think about using their great models to improve tableview management http://latest.docs.nimbuskit.info/TableViewModels.html I was about to start integrating it in my branch to get html property in labels, but i thought it was a better idea to first talk about it here. What do you think?
