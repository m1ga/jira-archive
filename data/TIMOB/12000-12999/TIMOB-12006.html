Description

This is NOT a 3.0.X bug -- it's very likely to have been introduced with changes for TIMOB-11282. See TIMOB-8708 for the test case here. Steps to reproduce: (1) Modify the app in TIMOB-8708 so images are 200 x 200. (2) You'll need a large number of contacts, 100 is good, probably 40 would be enough. (3) Run the app. Kick scroll to move the table quickly. You need to move the table fast enough that a few screens scroll by. (4) Notice that images shown are now blank. Actual results: Images are blank Expected result: Images show pictures of contacts
