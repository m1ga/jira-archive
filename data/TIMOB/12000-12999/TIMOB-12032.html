Description

The module API reference includes runOnUiThread for the @Kroll.module and @Kroll.getProperty, setProperty annotations. However, support for this was as far as I know removed in 1.8. For reference, see point #18 under Specific Changes, below: https://wiki.appcelerator.org/display/guides2/Android+Module+Porting+Guide+for+1.8.0.1#AndroidModulePortingGuidefor1.8.0.1-Specificchanges
