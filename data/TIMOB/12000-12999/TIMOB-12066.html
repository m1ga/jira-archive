Description

Test steps: 1. Install KS (in our case it is an App with custom info.plist with version #) from Studio on device, make sure there is no KS previously installed on device. 2. Change SDK build in tiapp.xml, save it and install on device. There will be an iTunes dialog window, asking to replace an existing App. Please, do. 3. Go to iTunes device's Apps. Actual result: After step 3 NO automatic "resync" happened, and there is "Remove" button next to the App instead "Update". The only way to replace an App is to delete it from device and install it from iTunes again. Expected result: After step 3 the button "Update" should be next to the App, so the App could be updated.
