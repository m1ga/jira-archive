Description

Setup the CI machine with the BB10 NDK detailed instructions at https://github.com/appcelerator/titanium_mobile_blackberry#how-to-get-started-with-titanium-blackberry-development Run nightly after the CI builds has happened:

Change directory /tmp execute "git clone --recursive https://github.com/appcelerator/titanium_mobile_blackberry".

To create the the blackberry drop-in, from the bb cloned repo change directory to cli/commands and then execute "./create_sdk" this will create a build folder with the blackberry sdk folder in it (see 4 below).

Grab from the repo build/sdk/ the blackberry folder and copy it to the latest CI sdk build put it at the same level as android, iphone, mobile web. The folder should go in OSX, Linux, Windows
