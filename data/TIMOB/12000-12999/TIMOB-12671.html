Description

The follow code does not fill the whole window. Using a view attached to the root window and filling it works fine. 
var win = Ti.UI.createWindow({
	backgroundColor: 'green',
    width: Ti.UI.FILL,
    height:Ti.UI.FILL
});

 
var view = Ti.UI.createView({
	backgroundColor: 'red',   
    width: 100,
    height: '100'
});

win.add(view);

win.open();
