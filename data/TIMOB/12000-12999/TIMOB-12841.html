Description

When using the default pins for map annotations, as the Android device resolution increases, the pin size decreases to the point that you can almost not select the pin annotation. You can see this clearly on a Nexus 4. Attached project runs OK on iPhone and Android HVGA AVD. On an Android WVGA AVD, the pins are OK but starting to get small. On the Nexus 4 yoou can barely select a pin. Google Maps on the same Nexus 4 has pins that are easy to select. Screenshots attached.
