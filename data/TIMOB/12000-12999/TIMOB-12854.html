Comments

  1. Josh Roesslein 2013-04-09 Indicators only support a "preferred" width and height. It will try to match the size provided the best it can.
  2. Josh Roesslein 2013-04-09 Setting no width or height should cause the indicator to use the smallest supported size. There is also a limit on how large the indicator can grow. If you set the size larger than the biggest supported indicator, the indicator will be centered within this bounds.
