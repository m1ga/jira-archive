Description

There have been requests for a number of features in the code processor that are syntactic in nature, but the Code Processor can really only do semantic analysis. There have also been requests to do analysis on non-js files, such as analyzing images, etc. To accomplish this, we can change the code processor to be a suite of tools, one of which is dynamic analysis consisting of the existing code processor.
