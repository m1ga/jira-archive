Description

Description: 1.Update to the latest sdk from the QNX momentics 2.Go to 'bbndk-env.sh' for mac osx & 'bbndk-env.bat' for windows in the bbndk folder (Normally located in applications/bbndk for mac & c:/users/--name--/bbndk for windows) 3.Open it with an editing program 4.Change the 'target_XXXXX' in QNX_TARGET="/Applications/bbndk/target_XXXXX/qnx6" to the latest version you downloaded 5.Save the file 6.Run the app for device/simulator 7.The app is built & runs on the simu/device fine Note: If these steps are not followed the app errors out & fails to build because it still points to the old BB sdk in the file.
