I have several SDK's installed and trying to build app with different SDK's to find regressions. Comments

  1. Chris Barber 2013-06-05 The --sdk option is used to specify the SDK used to bootstrap the build. You should generally have the active SDK set to the most recent GA release, then you should set the SDK you want to build with via the value in the tiapp.xml. No matter what --sdk you specify, the build will always use whatever version is in the tiapp.xml.
