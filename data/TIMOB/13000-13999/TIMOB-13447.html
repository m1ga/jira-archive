  4. Patrick Seda 2013-04-08 Sample test case for replicating. Using the attached graphic "lilypad.png", the image rendered as a backgroundImage of the View will have pixelated edges in comparision to the ImageView. The issue will be noticeable on simulator but is more pronounced on retina devices. 
       var aWin = Ti.UI.createWindow({backgroundColor:'#fff'});
   var viewWithBG = Ti.UI.createView({
   	height:100, width:250, top:10, left:10,
   	backgroundImage:'lilypad.png'
   });
   var imageView = Ti.UI.createImageView({
   	height:100, width:250, top:10, left:280,
   	image:'lilypad.png'
   });
   aWin.add(viewWithBG);
   aWin.add(imageView);
   aWin.open();
