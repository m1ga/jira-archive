Make sure you have some events in the calendar.

Run the test app app.js

Expected: All Calendars from the native Calendar app should show up on the Picker. Actual: No Calendar is shown.

Sabil Rahim 2013-04-16 This is essentially happening because of the conditional compilation of iOS 6 code that we put in for enabling privacy feature on iOS 6 and above. and Only happens if you build using Xcode 4.4 and try installing on iOS 6 device. Solution for this would be to, go to > settings > privacy and manually re-enable access for the given app to calendars.
  3. Blain Hamon 2013-05-07 This is exactly why we should require running a current version of XCode, because Titanium cannot link to necessary commands if they're not there in an outdated version of the iOS SDK.
