Description

This is a regression. When python scripts are used to build the project the 'i18n' folder is copied to the assets folder but not with the CLI Description: 1. Create an app for BlackBerry which has locale functionality e.g the single window app from the default templates in studio 2. Build & run the app on BlackBerry simulator/device 3. Now go to the app project in studio & expand the build folder 4. Under assets folder check if folder named 'i18n' is present Actual Result: 1. The 'i18n' folder for localization does not get copied in assets in build folder Expected Result: 1. The 'i18n' folder gets copied in to the assets folder
