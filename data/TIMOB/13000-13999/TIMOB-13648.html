Description

Problem description

After modifying an Alloy project code after the first run, a red screen with Parse error is displayed. Also, errors in the log are always displayed.

Steps to reproduce

1. Create an Alloy project and run using LiveView on iOS 2. Modify a UI element (for example a label). Result: red screen with 'parse error' is displayed and parse errors are displayed in the console log 3. Click on dismiss Result: code is changed as expected 4. Modify again any UI element Result: no error message screen is displayed, however errors in the log are always displayed. Attaching screenshot and error logs.
