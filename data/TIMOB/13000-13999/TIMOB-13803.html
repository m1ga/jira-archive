Description

Setting the height property in a button resizes the container that contains it but not the button itself. Blackberry has hardcoded the height of the buttons. According to their documentation, the height cannot be changed. This is a huge problem for us. All the other platforms allow the developer to set whatever size they want (width/height) on a button. For parity reasons, we need to find a way to make this happen.
