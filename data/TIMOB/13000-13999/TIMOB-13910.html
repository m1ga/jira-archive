Description

The current versioning scheme for node-appc doesn't allow for multi-branch development like the other products do (e.g. there's no way to work on 3.2.0 and 3.1.1 at the same time). We need to bump to version 1.0.0 and increment the minor version to coincide with the SDK minor version increases. Note: all other projects that depend on node-appc will need to update their package.json dependencies to match.
