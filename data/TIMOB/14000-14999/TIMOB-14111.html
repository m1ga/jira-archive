Description

*Problem* I have a local HTML file that I would like to open in a webview in a window. The file contains HTML anchors and I would like the view to position itself to that anchor when it opens. *Note* It works on Android. The file opens in the webview positioned at the myanchorpoint anchor. *app.js* 
var win = Ti.UI.createWindow({
	backgroundColor : "#FFF"
});

var webview = Titanium.UI.createWebView({
	url : 'mypage.html#myanchorpoint'
});
win.add(webview);

win.open();
