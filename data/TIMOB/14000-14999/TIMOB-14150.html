Problem description

When trying to build a project using Xcode, if the target is iPhone v6.0 and below, the build fails as ti.cloud is not found.

Steps to reproduce

1) Include ti.cloud in tiapp.xml 2) Just require it in app.js 3) Build for iPhone Simulator 4) Open the project in Xcode and build choosing iPhone 6.0 or below The expected result would be a running app, but instead the following error is raised: [ERROR] Script Error Couldn't find module: ti.cloud Same code works fine using iPhone 6.1 in Xcode
