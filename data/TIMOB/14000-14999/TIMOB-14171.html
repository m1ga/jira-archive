Description

This is a regression. It was working fine in previous 3.1.1 builds. Description: 1. Create an app in studio for Blackberry 2. Make sure your device is on AppcelStage access point 3. Follow this wiki page for analytics setup & test info : https://wiki.appcelerator.org/display/tp/Analytics+Test+Plan 3. Run the app on BB device. 4. Observe that after the first app launch you will see the enroll & a start event 5. Now close the app & re launch it 6. Obeserve the events Actual Result: 1. We get enroll event on each launch efter the first launch Expected Result: 1. We should not get enroll event on each launch after first launch Note : The sid for the enroll events for each launch is the same which means these are duplicate enrolls.
