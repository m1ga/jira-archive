Comments

  1. Michael Xia 2013-06-18 Could you try the following? 1. Open Preferences > Studio > Troubleshooting page; 2. Change "Debug Level" to All and make the entry "com.aptana.core/debug/shell" checked; 3. Restart Studio and install on Android emulator or device again; 4. Open Help > Studio > View Log File and find the titanium build command toward the end of the file; 5. Run the same command from the command line to see if it produces the same error. Another thing to make sure is to open Run > Run Configurations dialog, select the Android emulator launch configuration for the project, and check if "Enable LiveView" is selected. Thanks.
