Description

Steps to Reproduce

1) Create a new project 2) Create a new Node.ACS service 3) Create a new Node.ACS method - it should be based on HTTP GET and return no value as part of the response 4) Import Node.ACS Bindings 5) make a call to the method via the Node.ACS binding created Note: See workaround in the image attached

Actual Result

Invoke function errors out since the call xhr.getResponseHeader returns undefined, preventing the callback function from being called.

Expected Result

InvokeService function should make the call to the node.ACS service and fire off the callback
