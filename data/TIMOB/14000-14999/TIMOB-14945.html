  4. Vishal Duggal 2013-08-27 Fixed by the window rearchitecture The translucent property on Ti.UI.Window was false by default. Now it is true by default for IOS7+ to match native behavior. The toolbar has no such default property so follows native behavior. Hence the UI difference. This has been fixed in master and will soon be backported to 3_1_X for 3.1.3
