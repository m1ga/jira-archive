Comments

  1. Pedro Enrique 2013-09-03 After some research, these are my findings. Custom fonts are supported on BB SDK 10.2.0 and above. We currently compile 10.0.9. This is not so much a problem, there is an internal macro that can be used to check for OS version. This could be used. According to the BB documentation, the custom fonts can only be used through QML - BB's declarative UI framework, and not programatically. This is the biggest limitation. More detailed information here: http://developer.blackberry.com/native/reference/cascades/bb__cascades__fontfacerule.html I will update this ticket as I come across more details.
