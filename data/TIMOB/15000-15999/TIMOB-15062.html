  1. Vishal Duggal 2013-09-06 This is not our issue but apple having an issue with posting notifications when an accessibility feature is in progress. If I modify the code to add a small delay before posting the notification, it works fine. 
       
   announceButton.addEventListener("click", function() {
       setTimeout(function(){
           Ti.App.fireSystemEvent(Ti.App.EVENT_ACCESSIBILITY_ANNOUNCEMENT,
                              "Please stand by for an urgent message from the Department of Home Land Security.");
       },500);
   });
