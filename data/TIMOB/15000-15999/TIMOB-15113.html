Description

Description of the problem

Opening a modal window with navBarHidden set to false does not show the navigation bar with the title

Steps to reproduce

Use the following code: 
var win = Titanium.UI.createWindow({
    title : 'Modal',
    backgroundColor : '#bbb'
});

win.open({
    modal : 'true',
    navBarHidden : false
});
The navigation bar does not show up. This worked in SDK 3.1.2 (both iOS6.1 and iOS7).

Comments

  1. Vishal Duggal 2013-09-12 This is a concious change. If you want a navigation bar in a modal window open a Ti.UI.iOS.NavigationWindow with modal true
