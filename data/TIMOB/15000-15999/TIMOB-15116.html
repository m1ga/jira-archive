Description

Description:

With the release of iOS 7, a new iOS simulator option is available via Xcode 5 GM seed. This option is iOS 7 only, iPhone 5 (4-inch 64-bit).

Steps to reproduce:

1) Install Xcode 5 GM seed 2) Install command-line tools 3) Open iOS simulator and navigate to hardware>device

Result:

The new simulator is available via the simulator menu but not configurable from iOS sim

Expected Result:

iOS sim supports all available options
