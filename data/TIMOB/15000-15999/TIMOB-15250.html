Description

Titanium apps should not show a soft menu button (at the bottom right of the screen) when run on Android 4.0 (ICS) and above, unless the window's activity.onCreateOptionsMenu callback has been set (implying that there is code to produce a menu for that window). Currently, every Titanium app shows the soft menu button on every window.
