Description

Comments

  1. Chris Barber 2013-09-25 In order to use the latest bleeding edge Titanium SDK 3.2 (master), you MUST use the bleeding edge Titanium CLI 3.2. (master). If the Titanium SDK 3.2 you are using is a CI build, it will work just fine. If you are building Titanium SDK 3.2 from source (using scons), you MUST specify the node-appc-branch=master param. 
       
    clean up npm cache
    
   sudo rm -rf ~/.npm
   
   
    get latest CLI
    
   sudo npm install -g git://github.com/appcelerator/titanium.git
   
   
    build titanium sdk from source
    
   cd titanium_mobile
   scons node-appc-branch=master
