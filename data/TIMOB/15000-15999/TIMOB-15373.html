Description

Impossible to set textid property after creation label. 
var win = Ti.UI.createWindow();

var lab = Ti.UI.createLabel({
	color: "red",
	text: "text: Hello"
});


//lab.text = L("hello"); // WORKS
lab.textid = "hello"; // DOESN'T WORK

win.add(lab);
win.open();
