Description

The CLI needs to validate app icons by making sure that the correct icons are provided for the selected deployment targets. For example if the iPad icon(s) are missing then the build should warn if iPad is specified as a deployment target. For iOS, the icon-to-deployment mapping rules are specified here: https://developer.apple.com/library/ios/documentation/iPhone/Conceptual/iPhoneOSProgrammingGuide/Introduction/Introduction.html#//apple_ref/doc/uid/TP40007072
