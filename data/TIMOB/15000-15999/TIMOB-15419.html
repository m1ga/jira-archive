Description

Android Emulator auto crashes for WebView

When I added web view in window. Its crash my android emulator. I have also tried for both local and remote web URL.

Test Code

var win1 = Titanium.UI.createWindow({
	backgroundColor : '#fff',

});

// Create a WebView
var aWebView = Ti.UI.createWebView({
	url : 'http://docs.appcelerator.com/'
});
 
win1.add(aWebView);

win1.open({
	fullscreen : true
});


<Alloy>
	<Window class="container" navBarHidden="true" exitOnClose="true">
		<WebView id="w1" url="http://volume8.logiamobile.com/login.aspx"></WebView>
	</Window>
</Alloy>

Steps to reproduce

Please follow my step for reproducing this issue. 1. Create new apps for Titanium Classic or Alloy 2. Replace my test code to app.js code 3. Now run in android emulator for testing 4. Look in your Android emulator for crash state
