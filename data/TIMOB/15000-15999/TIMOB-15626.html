Description

Steps to Reproduce

1) Create a new 2-tab Alloy project, and follow the guide as per http://www.webdevd.com/titanium-apps-with-android-action-bar-tabgroups-and-menu-items/ until just before Adding Menu Items 2) Export project as Android APK and install on device with screen size larger than 800 px on any one side (e.g. Samsung Galaxy S3/S4, Xperia V, etc.)

Actual Result

Shows the application splash screen on Android devices with screen size larger than 800 px on any one side

Expected Result

Should show the application windows and tabs on Android devices with screen size larger than 800 px on any one side
