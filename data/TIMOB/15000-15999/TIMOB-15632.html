Description

Description

Running the iPhone app into iPad mini or iPad (non-retina) which is on iOS 7 the status bar is not shown in the device. You should run the iPhone app into iPad (not ipad app).

Steps to reproduce

iPad mini or non-retina ipad (simulator or device) with iOS 7 and titanium sdk 3.1.3 GA install any simple iphone app (Where deployment target is selected as iPhone only) Run the app in iphone simulator -> this will launch the iphone simulator. In iOS simulator menu (top bar) -> select hardware -> device ->iPad then again launch the app again in iPad simulator.
