Description

Description:

When studio prompts users to install CLI 3.1.2, the changelog is displayed. Currently the changelog is only listing 3.1.1 as the latest entry. This might cause confusion.

Steps to reproduce:

1) Remove CLI with "sudo npm -g remove titanium" OR if you have a version earlier than 3.1.2, skip this step. 2) Launch studio and "check for appcelerator updates" 3) Examine the "release notes" (changelog) displayed

Result:

The latest entry is 3.1.1

Expected Result:

The latest entry is 3.1.2 and describes the fixes made.
