Description

*Details:* node-appc modules in SDK and CLI do not match. Specifically, /node_modules/node-appc is not a beta version where as /usr/local/lib/node_modules/titanium/node_modules/node-appc is a beta version *Step to reproduce:* 1. Get the beta CLI: sudo npm install -g titanium@3.2.0-beta 2. Get the latest 3_2_X SDK: ti sdk install -b 3_2_X -d 3. Go to /node_modules/node-appc/package.json and /usr/local/lib/node_modules/titanium/node_modules/node-appc/package.json *Actual:* Both versions do not match (see attachments) *Expected:* The SDK node-appc should be beta version and from the same source.
