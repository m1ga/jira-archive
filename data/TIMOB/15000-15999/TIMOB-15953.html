Description

The Tizen installation SDK location is stored at C:\Users\[user]\AppData\Local\tizen-sdk-data\tizensdkpath and Titanium SDK read this configuration file to store the Tizen SDK location. However, if there is a space in the config file path (like the [user] in my case is 'Praveen Innamuri'), then SDK fails to read the correct path from the registry and Tizen app fails to build/run.
