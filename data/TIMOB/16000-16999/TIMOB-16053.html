  1. Samuel Dowse 2013-12-20

    Extra Information

    You cannot upload the .appxupload file generated by the CLI build. However if you open the generated .sln file in Visual Studio and package to .appxupload it will successfully upload to the Windows Store.

    Steps To Reproduce

    1. Create a hybrid application 2. Package the application for Windows Store 3. Open the .sln file in the build folder in Visual Studio 4. Select "Project -> Store -> Create app packages" 5. Run through the packaging wizard 6. Upload the generated .appxupload file to the Windows dev center
  2. Ingo Muschenetz 2014-05-06 Note that you can work around this just by opening up the solution in Visual Studio and doing the packaging manually.
  3. Lee Morris 2017-03-20 Closing ticket as Windows Hybrid is no longer supported.
