Description

Using the RC stream for AppC Studio 3.2.x, I was testing existing projects by building the project with 3.1.3GA SDK. When you try to run iOS, you get this console error from the titanium CLI: http://screencast.com/t/igOTQZTg8Yl This happens on new mobile projects as well as existing projects you import. The goal is to have customers be able to run projects with 3.1.3GA SDK.
