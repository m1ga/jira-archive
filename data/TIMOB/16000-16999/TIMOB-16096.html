Description

The android audio recording module developed by Codeboxed is exactly what I'm asking to be included in the base software. This feature has been requested numerous times on the Q&A board for quite some time. The Codeboxed module has become outdated due to current Titanium releases, their devs ignore attempts to discover the progress of the module, and for all intents and purposes have abandoned the project. That is an example of the downside of a module market. Users become dependent on a module and then become handicapped due to lack of support. If you can clone the audio recording features of that module, and the ability to customize the user interface (without having to use intents), that would satisfy a sorely missing feature of the android-based titanium platform.
