  1. Davide Cassenti 2013-03-14 The rightNavButton property expect an object of type Ti.UI.View; this means that you can add any view to it, not only buttons. You can then create a wrapper view and put the two buttons in it: 
       var view = Ti.UI.createView({
   	layout: 'horizontal'
   });
   
   var btn1 = Ti.UI.createButton({
   	title: 'Btn 1'
   });
   
   var btn2 = Ti.UI.createButton({
   	title: 'Btn 2'
   });
    Hope it helps
