Description

Description

BlackBerry 10.2 can run Android APK files. Installing an APK built in Titanium Studio successfully installs. However the analytic data sent from the device is incorrect. It will send: OS: Android Platform: Android Model: Z10 It has correctly identified the model of the phone, but not the OS and Platform.

Steps To Reproduce

1. Create an Android application 2. Publish the application to an APK file 3. Install the APK onto the BlackBerry device

Expected Result

Analytic data should show a BlackBerry install

Actual Result

Analytic data shows an Android install
