Description

Simulator fails to launch if Xcode 5.1beta4 is selected. Application successfully runs on device and can be packaged, however simulator fails to install and never loads to the home screen.

Steps To Reproduce

1. Install XCode 5.1beta4 2. Open Studio and create a default project 3. Run the project on an iOS simulator

Expected Result

Application installs and runs on simulator

Actual Result

Application fails to install and simulator opens a blank screen
