Description

*Details:* On iOS 7, after deleting an item from a ListView, the border between the items will not appear. *This is not a regression and only occurs on iOS 7.* *Steps to reproduce:* 1. Create a Titanium classic app with the app.js sample code attachment or create an Alloy app with the index.js and index.xml sample code attachments 2. Install sample app to an iOS 7 device 3. Press on an item to delete it *Actual:* The border between the items will not appear (see missing_border.png). *Expected:* The border between the items should appear after deleting an item from the ListView.
