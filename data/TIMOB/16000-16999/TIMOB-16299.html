Description

Titanium developers have expressed an interest in nesting top level window managers (Tab hosting a NavigationWindow, NavigationWindow hosting TabGroup etc). While it is possible to do that there is no support for NavigationBar management in the WindowProxy object. The NavigationBar management is only exposed in the TiUIWindowProxy object. This ticket is a request to expose that functionality on the base WindowProxy object.
