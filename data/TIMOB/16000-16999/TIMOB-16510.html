Check that is in the element of AndroidManifest.xml The Google Maps v2 add-on module requires a Google API key to be set in your Android manifest (even for development and testing) and the Google Play services SDK, which can be installed using the Android SDK manager. This module is only supported by Titanium SDK 3.1.0 and later and will not run on the Android emulator. Google Play services is also required to be installed on the device.
