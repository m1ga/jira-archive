Description

Description:

While trying to upload a status using the sample ti.cloud app, the upload fails. See screenshot.

Note:

This is a regression. It does not occur with 3.2.2.GA This may be the same root cause as the linked ticket, so may be a duplicate

Steps to reproduce:

1. Create a default Titanium classic app 2. Drop in example project from ti.cloud example folder 3. Install and launch app to an iOS device 4. Go to Users and create a user and log in 5. Press Back button to get to main menu and select "Status" 6. Press "Create Status" 7. Enter a message in the "Message" field 8. Choose either Photo option and add a photo 9. Press "Create"

Result:

The status posting will fail. See screenshot

Expected Result:

The status posting succeeds and the photo plus status can be viewed on the dashboard
