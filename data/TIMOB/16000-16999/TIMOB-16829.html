Description

Description

Having analytics set to true in the tiapp.xml causes no error, and analytical data is sent and received properly. However, setting it to false crashes the application whenever an event is sent (e.g, featureEvent)

Steps To Reproduce

1. Download the attached project 2. Set analytics to false in the tiapp.xml 3. Run the project on blackberry device 4. Select featureEvent

Expected Result

Nothing should happen, no analytics should be sent and the app should continue running.

Actual Result

Application crashes and closes
