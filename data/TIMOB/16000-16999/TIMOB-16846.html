Description

Description

Windows 8.1 Developer Preview is available for developers. The Titanium CLI does not detect this new phone OS, therefore blocking the user from building Windows Hybrid applications to device

Steps To Reproduce

1. Install the latest 8.1 phone update 2. Run 'ti info' 3. Create and build a Windows Hybrid Application to the device

Expected Result

Phone should be detected, application should also be built to device.

Actual Result

Phone is not detected, cannot build to device.
