Description

Currently, manifest.json includes the git hash for just the commit in titanium_mobile. Ideally, every repo which makes up the actual build for the Titanium SDK should have the git hash in question listed as well. One suggestion is that we have some sort of list of organization/repo_name: hash for everything listed in https://github.com/appcelerator/titanium_releases/blob/3_2_X/titanium.json. So: * appcelerator/titanium_mobile * appcelerator/titanium_mobile_blackberry * appcelerator-modules/ti.facebook * appcelerator-modules/ti.cloud * appcelerator-modules/ti.cloudpush * appcelerator-modules/ti.map * appcelerator-modules/urlSession
