  3. Ritu Agrawal 2014-05-07 Moving this ticket to engineering as I can reproduce the issue with the provided test case. Setting customView property causes pin to disappear from the map. A typical use case is to show the pin and then custom view should be displayed when the pin is clicked but in this case pin is not drawn when the custom view is used. It is perfectly legal to use pincolor and customView together as per documentation. Only exception is an image i.e. an image cannot be used for the pin when a customView is used but setting the pincolor together with customView is permissible.
