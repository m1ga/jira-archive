Description

Problem Description

App hangs on the splash screen when ti.cloud module is added in the module section within tiapp.xml file.

Steps to reproduce:

1. Create a new application using titanium/appcelerator studio having the deployment target as mobileweb (alloy project), add the ti.cloud module when creating. 2. Changed the SDK version in tiapp.xml to the Hybrid SDK 3. Open the command prompt and move to the project directory 4. Build for device using the command - titanium build -p mobileweb -T wp8 -C xe --wp8-publisher-guid

Actual Result:

App hangs on the splash screen

Expected Result:

UI for the main screen should get opened.
