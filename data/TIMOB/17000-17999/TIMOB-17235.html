  12. ArnaudM 2015-10-01 I found something interesting : - If you add a "top" attribute to the labels in the example above, then the flickering disappears. (That's a workaround!) 
        var label1 = Titanium.UI.createLabel({
      color:'#999',
      text:'I am Window 1',
      font:{fontSize:20,fontFamily:'Helvetica Neue'},
      textAlign:'center',
      top: 100
    });
    - And if you just set a "bottom" attribute, then the label will go up and down before stabilizing on first open. The tabgroup seems to interfere with the window height.
