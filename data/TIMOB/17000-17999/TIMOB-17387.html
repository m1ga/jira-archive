Description

We supply the wrong size graphics for the iPad splash screen graphics. Our graphics omit the height of the status bar, which apparently is not correct. See https://developer.apple.com/library/ios/documentation/UserExperience/Conceptual/MobileHIG/IconMatrix.html for proper sizes. The correct graphic sizes are also noted in this link to the TiCons project: https://github.com/FokkeZB/TiCons-CLI/blob/master/lib/specs.js#L160
