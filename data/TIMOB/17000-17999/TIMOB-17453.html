Description

Enhance ListView with advanced features for the Financial Services Sector - Locked Column(s) & Horizontal Scrolling & Clickable Column Headers Financial Services apps and others make use of tabular data. Handheld devices naturally do not display tabular data well as there is little room to display large tables Several apps implement these advanced features natively (Etrade, TD ThinkOrSwim) in their iPhone apps. The attached videos & screen shots illustrate the features I am talking about: * Horizontal Scroll * Locked Column * Clickable/Sortable Column Headers While there is not a specific customer asking for this, we know people want better ways to view tabular data and we see apps out there that are doing this. I hope we can add these features to our platform
