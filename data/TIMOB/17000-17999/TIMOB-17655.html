Description

If the *Process Unvisited Code* option is selected in code processor preferences, It fails to run for platform iPhone for Alloy projects. (!) This is *not a regression* since it's occurring in 3.3.0 GA release as well

Steps To Reproduce

1. Create an Alloy project 2. Right click > Code Analysis > Preferences.. 3. Select the option *Process Unvisited Code* 4. Run the code analyzer for iPhone

Expected Result

Code Analyzer should generate report as usual

Actual Result

Code analyzer fails with error message. Please refer the attachment for more info.
