Description

Description

When a popover is displayed on screen and then the screen orientation is changed, the popover is positioned on the screen incorrectly. The issue does not appear on an iOS7 iPad, the popover is aligned correctly when orientation is changed.

Steps To Reproduce

1. Using the attached app.js build to an iOS8 iPad. 2. Tap show gallery and when the popover appears rotate the device.

Actual Result

The popover is not centered when rotated.

Expected Result

The popover should be centered when rotated.
