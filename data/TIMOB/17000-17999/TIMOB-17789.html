Description

The CLI gives us the option to turn off device auto-selection, after which you're prompted with a list of simulators on which to run your app. I'd like to be able to specify a default sim to be used when autoSelectDevice is set to true. (Currently, the first detected simulator type is used.) This would help when repeatedly testing an app on a specific target sim type. Something like: ti config ios.defaultSimulator
