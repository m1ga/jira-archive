  7. Chris Barber 2016-08-19 This may seem like a non-issue being that Tizen is dead, but actually, this is a bug. We only identify folders in the Resources directory as platform folders if that platform exists in the current SDK. Instead, we should just create a list of all known platform names that have ever existing and combine it with the known platforms in an SDK (which will be mostly duplicates). We'll get this fixed in the next rev of the tooling.
