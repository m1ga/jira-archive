Description

Issue description

With the Google Maps app installed and already launched if you attempt to use maps in a Titanium App using the 3.4.0.GA TiSDK and Android API Level 18 the map tiles never load. On Android L available at http://developer.android.com/preview/index.html# Running app on a Nexus 5 and a Nexus 7.

Steps to Reproduce

Build the attached app.js and launch it on a Android Device not running L

Notice that the map tiles load.

Install the same app on a device running Android L.

Notice that the map tiles do not load.
