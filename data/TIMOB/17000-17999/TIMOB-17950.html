Issue Description

Appcelerator Studio fails to build to simulator on Windows machine after completing installation steps.

Steps to reproduce

Go to File option menu > New > Mobile App Project > An alloy/classic app.

Let Appcelerator Studio enable services for the app.

Then build for AVD simulator under Run Configurations.

The build process collides on an Error.
