Description

Description

When using a picker, the selector can be positioned to be between the bottom two items in the picker, this appears to only be possible on a iPhone 6 Plus, as on a iPad Air 2, iPhone 6 and iPhone 5S the issue cannot be reproduced. *Notes* This *is not* a regression I am unable to recreate the issue using a picker in a native application

Steps To Reproduce

1. Run the attached app.js on an iPhone 6 Plus 2. Position the item selector between the bottom two items

Actual Result

The selector will stay between both items

Expected Result

The selector should move to an item above or below it
