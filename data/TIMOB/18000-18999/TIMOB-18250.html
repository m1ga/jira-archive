Description

Description

This is *is not a regression* When a tab is added to a tabgroup, the icon for the current tab is no longer highlighted as the active tab in the bar at the bottom of the screen.

Steps To Reproduce

1. Add the attached app.js to an existing project and build to a device 2. Tap 'Click me'

Actual Result

The current tab will no longer be shown as the active tab in the bottom of the screen

Expected Result

The current tab should be shown as the active tab in the bottom of the screen
