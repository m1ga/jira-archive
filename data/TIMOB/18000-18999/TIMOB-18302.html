Description

When retrieving a javascript Date object from App.Properties with Ti.App.Properties.getObject that has been saved with Ti.App.Properties.setObject, it is returned as a string on Android. *Example:* {noformat} var date = new Date(); Ti.App.Properties.setObject("date", date); var retrievedDate = Ti.App.Properties.getObject("date"); console.log(typeof retrievedDate); // On android this will echo "string" // On iOS this will echo "object" {noformat}
