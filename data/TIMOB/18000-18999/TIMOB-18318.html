Description

I just noticed that a TitaniumKit class Titanium::TiModule was introduced that duplicates the existing module Titanium::Module. The code from Titanium::TiModule should be moved to Titanium::Module and then Titanium::TiModule should be deleted. Existing Titanium::Module: [Module.hpp](https://github.com/appcelerator/titanium_mobile_windows/blob/master/Source/TitaniumKit/include/Titanium/Module.hpp) and [Module.cpp](https://github.com/appcelerator/titanium_mobile_windows/blob/master/Source/TitaniumKit/src/Module.cpp) Duplicate Titanium::TiModule: [TiModule.hpp](https://github.com/appcelerator/titanium_mobile_windows/blob/master/Source/TitaniumKit/include/Titanium/TiModule.hpp) and [TiModule.cpp](https://github.com/appcelerator/titanium_mobile_windows/blob/master/Source/TitaniumKit/src/TiModule.cpp)
