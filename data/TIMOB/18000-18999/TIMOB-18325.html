Description

TitaniumKit Module instance crashes as soon as its destructor is fired. 
javascript
'use strict';

for (var i = 0; i < 1000; i++) {
    var b = Ti.UI.createButton({
        top: 20,
        bottom: 20,
        width: 200,
        height: Ti.UI.SIZE
    });
    b = null;
}
This crashes at UI::Button destructor. Tested on Windows Store app, 100% reproducible.
