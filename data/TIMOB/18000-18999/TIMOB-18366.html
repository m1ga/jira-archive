Description

KitchenSink crashes when I add and remove tabs in TabGroups.

It's not a regression since same thing happens in 3.4.1 GA

Steps To Reproduce

1. Install the KitchenSink in your device 2. Go to Base UI > Tab Groups 3. Press *Add/Remove Tab* button 4. Press *Close/Animate Tab Group* button 5. Press *Add/Remove Tab* button again

Actual Result

KitchenSink crashes. Crash log is attached.

Expected Result

App should not crash. Note: It's crashing on iPhones not in iPads. That is, crash is observed when *more* tab is created in TabGroup.
