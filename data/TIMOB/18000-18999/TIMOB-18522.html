Description

Windows supports much more than iOS's app badges. We can do app badges in Windows, similar to Titanium.UI.iPhone.appBadge, but Windows supports "glyphs" in addition to numbers: https://msdn.microsoft.com/en-us/library/windows/apps/hh868225.aspx We can also do something that iOS cannot: change the tile image and/or set a message/text. This is equivalent to changing the appicon for iOS. See https://msdn.microsoft.com/en-us/library/windows/apps/hh868253.aspx
