Description

We need to add a new Windows only API for altering the app's tile. This involves setting new content using the available templates. Typically the templates will include a slot for an image and for text, and the templates are intended for different tile sizes (which the user has configured which of the sizes to use for your app). https://msdn.microsoft.com/en-us/library/windows/apps/xaml/hh868253.aspx
