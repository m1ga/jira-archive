Description

I was looking to use the autolink property for email addresses and phone numbers. I found out this only works on iOS with a TextArea which seems a bit odd. Any reason we cannot have autolink available for iOS on Labels and TextFields?

Comments

  1. Vishal Duggal 2015-02-26 Autolinking as a native property is only available on UITextView(Ti.UI.TextArea on the titanium platform). To support links on labels you can use the link event along with the AttributedString. There is no support for link events on TextField on iOS
