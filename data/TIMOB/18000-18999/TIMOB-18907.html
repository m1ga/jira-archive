Description

To set the bar-color for Windows in a Ti.UI.MobileWeb.NavigationGroup it seems that you have to use barColor on the Windows in the navigation group. This works for the initial window, but not for child windows you open in it later.

To reproduce

Run the new Alloy version of the Sample.RSS on MobileWeb: https://github.com/fokkezb/Sample.RSS/tree/alloy As you see in the [app.tss](https://github.com/FokkeZB/Sample.RSS/blob/alloy/app/styles/app.tss#L14) I have set the barColor to red on all windows (thus including child windows). But as you see when you run it only the initial window has a red bar.
