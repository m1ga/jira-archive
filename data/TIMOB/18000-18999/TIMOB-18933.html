Description

Since both Google and [we](http://docs.appcelerator.com/platform/latest/#!/guide/Icons_and_Splash_Screens-section-29004897_IconsandSplashScreens-Androidsplashscreenconsiderations) recommend using nine-patch for splash images, our templates should do so as well. The existing [Titanium](https://github.com/appcelerator/titanium_mobile/blob/master/android/templates/app/default/template/Resources/android/images/res-long-port-hdpi/default.png) splash images are out-dated anyway and if we the white "A" on an Appcelerator-red background it would work fine as a nine-patch image.
