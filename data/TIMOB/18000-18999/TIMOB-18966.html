Description

We removed login from the Titanium CLI and replaced most operations with a NOOP. However, the status command does not work anymore and scripts like [grunt-titanium](https://github.com/tonylukasavage/grunt-titanium/blob/master/tasks/titanium.js#L302-L312) that rely on it don't work as a result of that. The status command should be a NOOP as well and return a result as if logged in to make sure scripts that depend on it continue to work.
