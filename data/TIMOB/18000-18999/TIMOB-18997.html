Description

There's no implementation to support auto layout in LayoutEngine. auto layout behavior should be either FILL or SIZE depending on components described in [Transitioning to the New UI Layout System](http://docs.appcelerator.com/platform/latest/#!/guide/Transitioning_to_the_New_UI_Layout_System) (For example, Label should treat auto as same as specifying SIZE). ViewLayoutDelegate should inform how components should layout against auto.
