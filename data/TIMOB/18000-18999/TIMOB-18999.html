Description

*Description* 'cm', 'in' and 'dp' values are not calculated correctly. This is because the PPI is not set. *Example* 
var win = Titanium.UI.createWindow({
    backgroundColor: '#fff'
});

var view = Titanium.UI.createView({
    backgroundColor: 'red',
    width: '1in',
    height: '10dp'
});
win.add(view);

win.open();
