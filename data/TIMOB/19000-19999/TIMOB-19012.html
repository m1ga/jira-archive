Description

The Problem

If you have a two tab app, move in to landscape, and click the second tab, both windows are visible at once.

Reproduction

ti create
cd createdApp/
ti build -p windows -C 8-1-1 -G 00000000-0000-1000-8000-000000000000
// rotate to landscape
// click tab 2
