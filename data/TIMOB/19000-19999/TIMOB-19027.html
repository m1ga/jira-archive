Description

When creating a Windows platform project the ti.windows.publishername property is not present in the tiapp.xml, this causes the build to fail as described in TIMOB-19025. This property should be auto-populated upon project creation

Steps To Reproduce

1. Create a new project 2. Open the tiapp.xml

Actual Result

The ti.windows.publishername property is not defined

Expected Result

The ti.windows.publishername should be defined on project creation
