Description

Every android app we build includes a few [built-in themes](https://github.com/appcelerator/titanium_mobile/blob/master/android/templates/build/theme.xml): * Theme.AppCompat.Fullscreen * Theme.AppCompat.Translucent * Theme.AppCompat.Translucent.NoTitleBar * Theme.AppCompat.Translucent.NoTitleBar.Fullscreen * Theme.Titanium, which extends either Theme.AppCompat directly or Theme.AppCompat.Fullscreen depending on the setting in tiapp.xml. Strangely enough, it does not include a theme that would cover one of the most common use cases: hiding the Action Bar. Our [guide](http://docs.appcelerator.com/platform/latest/#!/guide/Android_Action_Bar-section-36735509_AndroidActionBar-HidingtheActionBar) explains how to do this using a custom theme, while it could be much simpeler if only we would include one by default. via *tiapp.xml*
