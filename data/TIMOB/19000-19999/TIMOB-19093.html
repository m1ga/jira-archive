Description

Description

When building for ws-local the build should pause to allow the user to import the certificate as the build will fail if they do not complete the workflow quick enough

Steps to reproduce

1. Build a project that does not have a pfx file 2. Create the pfx through the prompts 3. When prompted to import the cert do not do anything

Actual result

The build will continue and fail

Expected result

The build should pause allowing time for the user to follow the workflow
