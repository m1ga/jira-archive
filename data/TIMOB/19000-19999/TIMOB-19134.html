Description

Description

The architecture reported in the information banner on build is for the architecture of Node rather than the OS architecture. For example if I run 32-bit Node on 64-bit Windows then the following is reported 
 Operating System 
Name = Microsoft Windows 8.1 Enterprise N 
Version = 6.3.9600 
Architecture = 32bit 

CPUs = 2 


Memory = 3756511232
[This is a Node issue](https://github.com/joyent/node/issues/2862)

Steps To Reproduce

1. On a 64-bit Windows machine install 32-bit Node 2. Run an build comand to show the banner appc run -p android -T device --build-only

Actual Result

32-bit is shown as the architecture

Expected Result

64-bit should be shown
