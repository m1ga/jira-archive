Description

Feature:

Right Now, Ti.Android.Intent only gives the option to send a single url within the putExtraUri() function. We need the option to use the ACTION_SEND_MULTIPLE instead of ACTION_SEND constant in order to allow an Array of URLs to be passed instead of a single String value.

Usage example: 

var photos = [ firstImage.nativePath, secondImage.nativePath ];
intent.putExtraUri(Ti.Android.EXTRA_STREAM, photos);

Expected Result:

success intent call

Actual Result:
