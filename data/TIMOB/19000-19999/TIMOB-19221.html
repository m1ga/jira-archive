Description

We might want a new markup support for Ti.UI.Windows.CommandBar button items using <Items> just like [Titanium.UI.iOS.Toolbar](http://docs.appcelerator.com/platform/latest/#!/api/Titanium.UI.iOS.Toolbar). Would be look like this: 
<CommandBar platform="windows" ns="Ti.UI.Windows">
    <Items>
        <AppBarButton ns="Ti.UI.Windows" id="refreshButton"/>
    </Items>
</CommandBar>
