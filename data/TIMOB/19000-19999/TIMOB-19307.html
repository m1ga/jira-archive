Description

When navigating to and from a ListView that has a custom template, sometimes the view stays completely empty. This is a 100% reoccuring issue in our project when following these steps: 1. Open List View (content visible) 2. Navigate back (iOS Navigation bar back button) 3. Open List View (content not visible) All further trys except for the second one result in the content being visible. *This issue is only related to the newest version of the studio. Version 4.0 works fine.*
