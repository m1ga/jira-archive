Description

Description:

Since we are providing a disclaimer in the help, it is not necessary to include the "™" in the text of the Titanium CLI "ti create" command or in "help" or elsewhere.

Steps to reproduce:

1) Install SDK 4.2.0+ (master as of today's date) 2) Install Titanium NPM package with "sudo npm install -g titanium" 3) Create a new project with "ti create" - note the output OR 3) Get help with "ti create -h" - note the output

Result:

"Apple Watch™" appears in multiple places

Expected Result:

No "™" needed.
