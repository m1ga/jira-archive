Description

If Xcode needs a license to be accepted, you will get the following error: 
[ERROR] :  An error occurred during build after 15s 47ms
[ERROR] :  69
TRACE  | titanium exited with exit code 1
ERROR  | Error: ti run exited with error code 1
    at ChildProcess.<anonymous>
You will see this in the build log: [TRACE] Agreeing to the Xcode/iOS license requires admin privileges, please re-run as root via sudo. However, error "69" seems to be indicative of this specific problem. We should catch that error value and correctly prompt the user to accept the license agreement. If we want to be truly elegant, we can do "sudo xcrun cc" and prompt the user ourselves.
