Description

The following sample demonstrates that the app crashes when you use both the backgroundColor and cardCornerRadius property on a CardView. If you use cardBackgroundColor instead it works. 
var win = Ti.UI.createWindow();

var card = Ti.UI.Android.createCardView({
	top: 20,
	left: 20,
	right: 20,
	contentPadding: 20,

	// Remove and it won't crash
	cardCornerRadius: 20,

	// Rename to cardBackgroundColor and it won't crash
	backgroundColor: '#F00'
});

card.add(Ti.UI.createLabel({
	text: 'Hello, world!',
	color: '#000'
}));

win.add(card);

win.open();
