Description

The Android Ti.XML.Node implementation has an undocumented evaluate() method that takes an xpath string expression and evaluates it. This is not found in our docs whatsoever for the class. We should either document it and make it official API, or remove it. Removing it would likely also allow us to remove the jaxen-1.1.jar dependency since it appears that's the only place we use it.
