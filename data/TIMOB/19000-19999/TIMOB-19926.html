Description

Support proximity sensor. Ti.App.proximityState, Ti.App.proximityDetection and Ti.App proximity event. Maybe Windows::Networking::Proximity package does the sensor work. *Test Case* 
var win = Ti.UI.createWindow({ backgroundColor: 'red' });

Ti.App.addEventListener('proximity', function (e) {
    win.setBackgroundColor(e.state ? 'green' : 'red');
});
Ti.App.setProximityDetection(true);

win.open();

<windows>
  <manifest>
    <Capabilities>
      ...
      <DeviceCapability Name="proximity" />
      ...
    </Capabilities>
  </manifest>
</windows>
