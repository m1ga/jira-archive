  1. Angel Petkov 2015-11-13 Hello , we have looked at the problem and believe this PR should fix it. While the showPagingControl property is set to false inside the scrollableView you should no longer see the black footer. PR pending https://github.com/appcelerator/titanium_mobile/pull/7436. 
       var win = Ti.UI.createWindow();
   
   var view1 = Ti.UI.createView({ backgroundColor:'#123' });
   
   var scrollableView = Ti.UI.createScrollableView({
     views:[view1],
     showPagingControl:false
   });
   
   win.add(scrollableView);
   win.open();
