Description

Titanium.UI.createTempFile() creates a temp file without extension.

On line 53 in FileSystemModule.java, the following is executed:
File f = File.createTempFile("tifile", "tmp");

Should have been:
File f = File.createTempFile("tifile", ".tmp");
