Description

When you create 2 labels, first the "frontlabel" and then the "backlabel", it shows the backlabel on top of the frontlabel. To change this, you can give the backlabel the zIndex of (for example) 2, and the frontlabel a zIndex of (for example) 8. This works perfect in a clean window. But not in a window inside a tabgroup. See the project in the attached zip file for an example.
