Description

Titanium apps don't distribute with android SDK Tools revision 7 (the full updated sdk) becouse the apkbuilder stopped working under apple.
An another method of building is needed, one without the apkbuilder.

See the following links for other people with this problem:
http://developer.appcelerator.com/question/66741/apkbuilder-is-deprecated-titanium-developer-121-sdk-1411-mac-osx"> http://developer.appcelerator.com/question/66741/apkbuilder-is-depr...
http://developer.appcelerator.com/question/67641/apkbuilderbat-deprecated--see---help"> http://developer.appcelerator.com/question/67641/apkbuilderbat-depr...

How to reproduce?
run "android" in the "tools" directory, go to "Available packages", check everything in the list and install it.
Now try to distribute the application from titanium, this doesn't work. Also running in emulator gives errors.
