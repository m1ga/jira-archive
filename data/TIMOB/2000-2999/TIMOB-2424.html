Description

Extra data passed in a Push Notification is stripped before it get's to the app.
e.g. acme1 & acme2 are not readable:

{

"aps" : {
    "alert" : "You got your emails.",
    "badge" : 9,
    "sound" : "bingbong.aiff"
},
"acme1" : "bar",
"acme2" : 42

}
