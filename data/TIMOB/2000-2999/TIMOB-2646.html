Description

Developers would like to have the ability to add custom launch parameters to the android emulator such as setting a HTTP proxy. Here is one example of the parameters they would like to be able to include: 
  # start the emulator
        emulator_cmd = [
            self.sdk.get_emulator(),
            '-avd',
            avd_name,
            '-port',
            '5560',
            '-sdcard',
            self.sdcard,
            '-logcat',
            "'*:d *'",
            '-no-boot-anim',

          '-http-proxy',


          'http://127.0.0.1:8888',

            '-partition-size',
            '128' # in between nexusone and droid
        ]

  4. Chris Barber 2013-10-18 This is now in the new Android build and can be set via a CLI plugin hook. Someday we may expose it via the tiapp.xml. To test, in your project, create the directory /plugins/test/1.0/hooks/test.js, then put the following in it: 
       exports.cliVersion = '>=3.2';
   
   exports.init = function (logger, config, cli) {
   	cli.on('build.android.startEmulator', {
   		pre: function (data, next) {
   			logger.log('[FUNCTION-PRE-HOOK] build.android.startEmulator'.magenta);
   			data.args[1].scale = '.25';
   			next();
   		}
   	});
   };
