Description

Android SDK: Titanium Developer, Run Emulator, "Screen" menu shows "Loading..." while "SDK" menu displays OK.

See following screenshot:

http://img267.imageshack.us/img267/7449/screenshot155i.png" alt= "screenshot1">

Also, see hi-res screenshot http://img513.imageshack.us/img513/823/screenshot155.png">here

The problem exists when the exact android packages are installed as in the screenshot. This has been proven a problem on Windows 7 and Ubuntu (by me) and Mac OSX 10.6.5 (by dtoth).

Workaround that resolves this completely is to close TiDev, install the package, "SDK Platform Android 1.5" and start TiDev and launch the app.

  2. hal 2011-04-15

    For users who may be confused by the workaround, "SDK Platform Android 1.5" can be installed using the Android SDK and AVD Manager tool (shown in the top-left of the screenshot above):

    • close all emulators
    • close Titanium Developer
    • install the "SDK Platform Android 1.5..." package
    • delete existing AVDs using the same tool as you installed the package
    • start Titanium Developer
    • launch app
