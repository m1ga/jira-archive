Description

This is not a regression from 5.1.0.GA but not sure when it broke.

Steps To Reproduce:

1. Create a classic/alloy app i appc studio. 2. Build it to genymotion emulator. 3. Observe when genymotion emulator launches.

Actual Results:

1. You will see that along with the genymotion emulator the virtual box app also gets launches automatically. *NOTE: I am not sure if this is an issue or something changed in genymotion which is causing this issue.*

Expected Results:

1. Only genymotion emulator should be launched.
