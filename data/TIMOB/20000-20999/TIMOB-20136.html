Description

Using this code: 
window = Ti.UI.createWindow({
	
});

password = Ti.UI.createTextField({
	passwordMask  : true,
	value : "this is a secret",
	editable : false,
	top : 100,	
	width : Ti.UI.FILL
});

switcher = Ti.UI.createButton({
	title : "Editable: "+password.getEditable(),
	top: 200
});

switcher.addEventListener("click",function(e){	
		password.setEditable(!password.getEditable());			
		switcher.setTitle("Editable: "+password.getEditable());
});

window.add(password);
window.add(switcher);
window.open();
The textfield "password" don't mask the content if field is not editable. If property editable is set to true, the value is masked. Works fine on iOS devices.
