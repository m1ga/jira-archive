Description

Ti.UI.WebView on Windows doesn't support Titanium API in the HTML as known as "local script" as described in the [API doc](http://docs.appcelerator.com/platform/latest/#!/api/Titanium.UI.WebView). Especially Ti.App.addEventListener, Ti.App.fireEvent and Ti.API logging functions are not available. See also: https://wiki.appcelerator.org/display/guides2/Communication+Between+WebViews+and+Titanium
