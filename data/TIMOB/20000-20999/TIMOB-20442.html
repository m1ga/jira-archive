Description

The releases on GitHub are marked as pre-release since 5.0.2.GA, which makes 5.0.1.GA still marked as latest release: https://github.com/appcelerator/titanium_mobile/releases

Comments

  1. Chee Kiat Ng 2016-03-02 This just requires a fix in Jenkins to identify if it's a pre-release or release, and release accordingly.
  2. Chee Kiat Ng 2016-03-11 Added a if else statement in shell script in jenkins: 
       if [ $TAG = "GA" ]
   then
      RELEASEFLAG="release"
   else
      RELEASEFLAG="prerelease"
   fi
   
   curl -u $APIKEY --header "Accept: application/vnd.github.manifold-preview" --data "{\"tag_name\":\"$TAGNAME\",\"target_commitish\":\"$GITHASH\",\"$RELEASEFLAG\": true}" "https://api.github.com/repos/appcelerator/titanium_mobile/releases"
  3. Eric Merriman 2016-05-12 Closing as implemented.
