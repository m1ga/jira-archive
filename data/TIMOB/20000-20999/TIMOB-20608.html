  10. Chris Barber 2017-04-27 f(), never f( ) f(a, b), never f( a, b ). However, for arrays, add the space: f([ a, b ]). if (true) {, never if(true){ } else {, never }\nelse {, though there's a ton of this going on. function (), never function() And most importantly, tabs, not spaces! :)
