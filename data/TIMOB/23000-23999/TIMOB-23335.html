Description

SDK 5.4.0.v20160426000033 Setting fontSize on a TextField in Windows Phone does not work, it keeps the default size no matter what is set. See example: index.xml 
<Alloy>
	<Window class="container">
		<TextField id="number" hintText="hint" keyboardType="Titanium.UI.KEYBOARD_DEFAULT"></TextField>
	</Window>
</Alloy>
index.tss 
'#number':{
	font: {
		fontSize: 50
	}
}
