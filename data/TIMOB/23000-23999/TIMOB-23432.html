Description

Android N adds support for displaying more than one app at the same time. On handheld devices, two apps can run side by side or one on top of the other in split-screen mode. Android N also adds new features to support apps that run in multi-window mode. We can implement said features. https://developer.android.com/preview/features/multi-window.html
