Description

All of the values in TableViewRow height and width are always 0. Test case: 

<TableView id="table">
  <TableViewRow id="row> 
      <Label> Hello world! </Label
  </TableViewRow>
</TableView>

$.table.addEventListener('postlayout', function (e){
    Ti.API.info("# Row height: " + $.row.rect.height);
});
Log result:

Row height: 0
