Description

Problem

All the methods from alloy (onClick, onBlur, onReturn, etc.) are returning a different e.source.apiName on Windows 8.1 & 10 then iOS & Android. Windows is returning Titanium.UI., while iOS & Android return Ti.Ui.

Expected

Windows perform the same as iOS & Android.

Sample Code

function doClick(e) {
    alert(e.source.apiName);
}

$.index.open();

<Alloy>
	<Window class="container">
		<Label id="label" onClick="doClick">Hello, World</Label>
	</Window>
</Alloy>

".container": {
	backgroundColor:"white"
}

"Label": {
	width: Ti.UI.SIZE,
	height: Ti.UI.SIZE,
	color: "#000"
}

"#label": {
	font: {
		fontSize: 12
	}
}

In ios/android you (Appcelerator) use the string "Ti", in windows you (Appcelerator) use the string "Titanium".
