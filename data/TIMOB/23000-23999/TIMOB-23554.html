Description

On Android, Ti.apiName returns 'Ti.Module', but should return 'Ti'. 
it('apiName', function () {
		should(Ti).have.readOnlyProperty('apiName').which.is.a.String;
		should(Ti.apiName).be.eql('Ti'); // equals 'Ti.Module' on Android, which is incorrect.
	});
