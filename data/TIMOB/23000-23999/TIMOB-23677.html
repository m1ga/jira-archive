  9. Eric Merriman 2016-10-19 Hello all, Confirmed. We have two ways that you can push using FCM: 1) The method described by [~aislam] above - this is the simplest. Simply create a new project in Firebase console, and use the "Server key (legacy token)" as "GCM API Key". Then add the "SenderID" to the "GCM Sender ID" and you can push. Of course you must follow the other steps listed to create a mobile app and create and register users, subscribe for push, etc.. 2) You can import a project into Firebase. Basically similar, the end result will be a new key is generated. The steps for this are: 2.1) Log into Firebase console 2.2) Choose "Import Google Project" from the project menu 2.3) Select "Project Settings" from the gear menu 2.4) Select the "CLOUD MESSAGING" Tab and use the key and sender ID as previously described If you return to the Older google API console, you will see two new keys are added to the project.
