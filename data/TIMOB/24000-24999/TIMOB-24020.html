Description

As many third party Frameworks as well as CocoaPods require shell scripts to be run during builds we should support adding those scripts to the Build Phases in the generated Xcode project. There are two different use cases for this: * Allow a user to define custom Shell Script Build Phases in appc.js, see TIMOB-23854 * CocoaPods adds various Shell Script Build Phases to copy pod resources and to embed frameworks into the project, see TIMOB-20565, TIMOB-23780 and TIMOB-23570
