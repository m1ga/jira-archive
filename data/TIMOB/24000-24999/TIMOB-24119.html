Description

Currently android.location is being used in the SDK. However, as Android has specified in the [documentation](https://developer.android.com/reference/android/location/package-summary.html): *This API is not the recommended method for accessing Android location.* The recommended way is by using [com.google.android.gms.location](https://developers.google.com/android/reference/com/google/android/gms/location/package-summary). Main reason: higher accuracy
