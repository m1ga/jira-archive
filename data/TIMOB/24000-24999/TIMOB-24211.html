Description

In 6.0.0, we supported the padding value for the TextField and TextArea, but the TextArea one was not implemented for iOS so far. This ticket should change that.

Comments

  1. Hans Knöchel 2016-12-12 PR: https://github.com/appcelerator/titanium_mobile/pull/8678 Test-Case: 
       var win = Ti.UI.createWindow({
       backgroundColor: '#fff'
   });
   
   var field = Ti.UI.createTextField({
       top: 40,
       width: 200,
       height: 40,
       backgroundColor: "#eee",
       padding: {left: 20, right: 20}
   });
   
   var area = Ti.UI.createTextArea({
       top: 200,
       width: 200,
       height: 200,
       backgroundColor: "#eee",
       padding: {left: 20, right: 20, top: 20, bottom: 20}
   });
   
   win.add(field);
   win.add(area);
   win.open();
