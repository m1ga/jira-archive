Comments

  1. Hans Knöchel 2017-03-01 External libraries are not part of the SDK. There is an open source module for iOS and Android to use the Firebase SDK in Titanium: https://github.com/chrisribe/ti-firebase. Example usage for this use-case: 
       var Firebase = require("ti.firebase");
   Firebase.configure();
   
   Firebase.FIRAnalytics.logEventWithName({
       name: 'xyz',
       parameters: {
           VALUE: 0.99,
           CURRENCY: 'EUR'
       }
   });
