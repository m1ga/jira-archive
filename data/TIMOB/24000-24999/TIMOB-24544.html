Description

Some customers use Hyperloop only on Android (or iOS), but setting the property "run-on-main-thread" will enable it for both, causing possible issues that are not fixed so far. So besides fixing them, the customer should be able to enable it per platform, e.g. using the platform="android" flag. This is currently blocking their release, so we should come up with a solution for them.
