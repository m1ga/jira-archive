  5. Jan Vennemann 2017-05-08 [~hansknoechel], the crash is caused by the ti.dynamiclib hook. It extends the Array prototype by just assigned the last property. This causes all arrays to have the iterable property last which causes the crash. To avoid that, the prototype extension should be defined like this: 
       if (!Array.prototype.last) {
     Object.defineProperty(Array.prototype, 'last', {
       value: function() {
         return this[this.length - 1];
       }
     });
   }
