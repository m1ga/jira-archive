Description

Alert Dialog has had tintColor since iOS 8+: 
UIAlertController *alertController = ....
[alertController.view setTintColor:[UIColor greenColor]];
Would it possible to implement this in Titanium?

Comments

  1. Hans Knöchel 2017-06-16 You can use the Ti.UI.setTintColor('green') method / computed property for setting a global tint-color already, but I see the point of using it for the alert specifically as well. I assume myAlert.tintColor = 'green' does not work?
  2. Hans Knöchel 2017-06-16 PR: https://github.com/appcelerator/titanium_mobile/pull/9150 Test-Case: 
       var win = Ti.UI.createWindow({
       backgroundColor: '#fff'
   });
   
   var btn = Ti.UI.createButton({
       title: 'Show Alert'
   });
   
   btn.addEventListener('click', function() {
       var alert = Ti.UI.createAlertDialog({
   		title: 'My Dialog',
   		message: 'Hello World',
   		buttonNames: ['OK'],
   		tintColor: 'red'
   	});
   	alert.show();
   });
   
   win.add(btn);
   win.open();
  3. Joseph Sachs 2017-06-16
