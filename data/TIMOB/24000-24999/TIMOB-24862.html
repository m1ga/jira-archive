  4. Jan Vennemann 2017-07-06 [~jerodfritz], Hyperloop 2.1.2 was released together with SDK 6.1.1.GA and it should have been automatically downloaded when the CLI checked for updates. If not you can manually force a update check by running appc config set lastUpdateCheckTiDownloads null and then re-running your build. Also make sure to update any version constraints you may have set in you tiapp.xml for the Hyperloop module and plugin.
