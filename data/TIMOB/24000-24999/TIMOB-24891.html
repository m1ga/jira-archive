Description

When running ti setup check under Android Environment the following is output, to my knowledge the android executable is now deprecated by Google, we should not spit out an error message like this for a deprecated tool (as long as their versions indicate it is deprecated I guess) 
  ×  android            "android" executable not found; please reinstall Android SDK

Steps to reproduce

Run \[appc\] ti setup check

Actual

× android "android" executable not found; please reinstall Android SDK

Expected

A warning that is less worrisome, or no output at all if it is no longer relevant to the tools
