Description

The Ti.UI.iPhone namespace components were deprecated between 3.5.0 and 5.2.0. It is time to remove it in 7.0.0 to speed-up the compile time and use all iOS-only API's on the Ti.UI.iOS namespace instead. Basically, since all API's have been moved already (including Alloy), there is no migration issue as well.
