  1. Vijay Singh 2017-08-07 [~vvazquezmontero] Keyboard types are already supported in titanium as you can see in document [here](http://docs.appcelerator.com/platform/latest/#!/api/Titanium.UI.TextArea-property-keyboardType) . Please see the following example for usage of keyboard type - 
       var win = Ti.UI.createWindow({
       backgroundColor: '#fff'
   });
   
   var textArea = Ti.UI.createTextArea({
       top: 150,
       width:200,
       value: 'Keyboard Type test',
       borderColor: 'red',
       backgroundColor: 'gray',
       keyboardType: Titanium.UI.KEYBOARD_TYPE_NUMBERS_PUNCTUATION
   });
   
   
   win.add(textArea);
   win.open();
    I think this is what customer is looking for.
