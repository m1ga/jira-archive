Description

With the introduction of dynamic frameworks in modules (TIMOB-20557) and the updated framework handling for Hyperloop (TIMOB-23570 and TIMOB-23853) the need to manually configure frameworks is no longer required. The use case to directly use Objective-C and Swift source files should also be discouraged in favor of using frameworks. For this to happen we should deprecate the thirdparty setting in the appc.js and remove it in Hyperloop 3.0.
