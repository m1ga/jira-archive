Description

Now that Alloy uses Babel to transpile ES6 code, we should update [kitchensink-v2](https://github.com/appcelerator/kitchensink-v2) to use ES6 as well. It is our most important sample-project and should reflect the latest technologies we provide. Changes to consider - Change var to let / const - Use arrow-syntax (() => instead of function() callback). Maybe even use promises? - Use classes instead of functions? - Add linting of possible?
