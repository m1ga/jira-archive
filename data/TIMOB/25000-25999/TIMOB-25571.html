  5. Hans Knöchel 2018-03-05 *EDIT*: Here is the real issue: The real issue is still CocoaPods, but a different one: When using frameworks, some can be Bitcode-enabled and some not. CocoaPods may not know how to handle that, so in the native world you would configure some frameworks implicitly to disable Bitcode, which Titanium does as well. Long story short: For some of them, it cannot detect it (when frameworks are managed by CocoaPods), so adding the following snippet to the end of your Podfile solves it (tested successfully with the Hyperloop Sample App): 
       post_install do |installer|
     installer.pods_project.targets.each do |target|
       target.build_configurations.each do |config|
         config.build_settings['ENABLE_BITCODE'] = 'NO'
       end
     end
   end
    Also added a note to the [Hyperloop Docs#CocoaPods](https://wiki.appcelerator.org/display/guides2/iOS+Hyperloop+Programming+Guide#iOSHyperloopProgrammingGuide-CocoaPods)
