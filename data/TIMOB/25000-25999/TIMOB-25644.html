Description

Hi guys, I think many of developers who uses newer phones has to check, if device has physical or software back button, menu button and home button. If they are software buttons, I have to calculate screen size - height of this buttons. It will be very nice to have property to check for this in native Titanium without Hyperloop. I check it like this: 
var keyCharacterMap = require('android.view.KeyCharacterMap');
var keyEvent = require('android.view.KeyEvent');
var hasBackButton = keyCharacterMap.deviceHasKey(keyEvent.KEYCODE_BACK);
I tested with Nexus 5X (software buttons, hasBackButton = false) and Samsung Galaxy A5(physical buttons, hasBackButton = true). Thanks in advance for response
