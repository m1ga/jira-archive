Description

*Summary:* The default splash image (and other images) appears skewed when running on iOS 8. This is not an issue on iOS 9 and newer OS versions. This issue happens in Titanium 6.1.1 and 7.3.0. *Steps to reproduce:*

Create a new "Classic App" project.

Builds and run it in the iOS 8 simulator.

Observe the default splash when displayed onscreen. (It appears skewed.)

Observe the tab icons at the bottom of the app. (They appear skewed.)
