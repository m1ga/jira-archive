Description

In our Ti.Geolocation docs we refer to three ways Android can use Geolocation: 1. Simple 2. Manual 3. Legacy The "legacy" mode seems deprecated since Ti SDK 2.0, so we should remove it together with it's deprecated ACCURACY_* constants and the frequency + preferredProvider property. Also, we should revise the Ti.Geolocation for Android in general to be sure we are up to date.
