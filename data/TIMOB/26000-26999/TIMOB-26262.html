Description

Titanium currently uses revision 27 of the Google Support libraries. We should update to revision 28.0.0 once it has been released by Google. (Currently in beta.) Revision 28 contains new features such as: * BottomAppBar * Chip and ChipGroup (To be used by Ti.UI.ButtonBar.) * MaterialButton * MaterialCardView * New theme called: Theme.MaterialComponents The release of this library will coincide with Google's release of Android P.
