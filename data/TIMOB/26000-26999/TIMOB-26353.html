Description

Dependencies that use ES6+ are currently not handled properly in Titanium. To reproduce: 1. Create a new Ti app 2. Run npm init in app/lib 3. Run npm i redux --save (in app/lib as well) 4. Add import { createStore } from 'redux' to your app.js / index.js (ES6 enabled) 5. Run your app
