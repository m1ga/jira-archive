H5.Description The jsca file generated during the build process appears to missing the Windows specific namespaces from the Ti.App and Ti.UI namespaces. This can be seen by opening up the api.jsca file under an SDK and searching for "Titanium.App.Windows" or "Titanium.UI.Windows", neither exist. I don't believe this to be a platform specific issue as the iOS namespaces appear in the jsca in the Windows zip. This causes problems for any products that make use of the jsca file (Studio, Atom plugin)
