Description

We need to override the Ti.UI.Slider track- and tint-colors which is already works for the Ti.UI.ProgressBar. Example: 
var win = Ti.UI.createWindow({
    backgroundColor: '#fff'
});

var slider = Ti.UI.createSlider({
    width: 200,
    min: 0,
    max: 1,
    value: 0.5,
    tintColor: 'red',
    trackTintColor: 'green'
});

win.add(slider);
win.open();
