Description

Jenkins modules builds were mysteriously failing suddenly on a new build node. Turns out I no longer was installing Android SDK build tools 25.0.3 but out pipeline library to build native modules was specifying to use that version. The error message it displays in this case is misleading and partially incorrect. It indicates that Builds tools 25.0.3 is not installed (correct!), but then summaries the various tool components versions that are installed and states build tools 25.0.3. is currently installed there (incorrect!). Please see the screenshot for the exact error.
