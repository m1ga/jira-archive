Description

https://developer.apple.com/design/human-interface-guidelines/ios/visual-design/color/#dynamic-system-colors Apple has added Dynamic System Colors that adapt to both light and dark UI modes. iOS defines two sets of background colors—system and grouped—each of which contains primary, secondary, and tertiary variants that help you convey a hierarchy of information. In general, use the grouped set of background colors when you have a grouped table view; otherwise, use the system set of background colors.
