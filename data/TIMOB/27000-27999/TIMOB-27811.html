Description

We have a reusable template library called template-kit (https://github.com/appcelerator/template-kit). We also have a bunch of Titanium templates that template-kit can install (https://github.com/appcelerator/titanium-templates). template-kit will be integrated into titaniumlib and used by the Titanium CLI v6 and Titanium appcd plugin. However, it would be beneficial if the Titanium SDK was refactored to use template-kit directly or use titaniumlib's template-kit wrapper. This way template-kit (and possibly titaniumlib) would have more testing and users would have access to the new templates.
