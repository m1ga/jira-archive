Description

The active tab is not highlighted. It does have the focus however an underline to highlight the active tab is missing. This is is regression since 9.0.0.GA. It works fine in 8.3.1.GA Steps to reproduce: 1. Create a default classic app 2. Run it on an android device/emulator Actual Result: The active/selected tab is not highlighted. Check the attached images for difference between 8.3.1.GA and 9.0.X results. Expected Result: The active tab should be highlighted.
