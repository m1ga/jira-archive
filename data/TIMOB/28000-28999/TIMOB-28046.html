Description

Update "titanium_mobile/android" to compile for Android 11 (ie: "compileSdkVersion" 30). This is needed in order to use the newest API Level 30 features so that we can work on the "Android 11" epic. Starting off, Titanium should not target API Level 30 by default until all Android 11 breaking changes have been resolved. But we should allow the developer to opt-in to targeting API Level 30 to test the newest features/changes.
