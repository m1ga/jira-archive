Description

- Setting a border will fill the view with borderColor if a backgroundColor is not specified *TEST CASE* 
const window = Ti.UI.createWindow({
	backgroundColor: 'white'
});
const border = Ti.UI.createView({
	width: 200,
	height: 200,
	borderRadius: 5,
	borderColor: 'gray',
	borderWidth: 3
});

window.add(border);
window.open();
