Description

Epic tracking all tasks needed to transition Titanium to material design. *Breaking-Change:* Material based widgets require a "Theme.MaterialComponents" based theme or else an exception will occur. So, apps using a custom theme based from "Theme.AppCompat" or similar will crash. This is normal and the exception will state that the theme will need to change.
