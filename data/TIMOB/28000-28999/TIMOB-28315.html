Description

This is a regression from 9.3.0.GA onwards. Not seen with SDK 9.2.2.GA

Steps to reproduce:

1. Use the code below in your app.js: 
const win = Ti.UI.createWindow({ backgroundColor: 'gray' });

const rows = [];
for (let i = 0; i < 100; i++) {
  rows.push({ title: Row #${i} });
}

const tableView = Ti.UI.createTableView({ data: rows });

win.add(tableView);
win.open();
2. Build for android device & let the app launch. 3. Press back button to close the app.

Actual results:
