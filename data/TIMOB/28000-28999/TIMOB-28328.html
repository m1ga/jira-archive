Description

Currently if the webpack plugin isn't installed then the SDK will ask the user to install via npm and perform an appcd restart. With daemon 4 there is a new appcd pm command available so we should direct users to use this instead. We probably want to see if we can sniff the daemon version from the request we make and maintain the npm message for 3.x and use appcd pm for 4.x
