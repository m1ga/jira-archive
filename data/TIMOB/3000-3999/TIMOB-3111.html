Description

The height of a search bar cannot be changed if added to a tableview on the droid platform through the search property. This is ok for medium density devices but on high density devices the search bar looks very tiny by default and since the height can not be changed it's an eye sore. You'll notice on the first tab the search bar is tiny, second tab it looks ok (not added added to tableview), third tab its still tiny even though we try to change the height.
