Description

Not new to 1.6.1

  1. Xcode 4 + 1.6.1
  2. create or have a simple mobile project
  3. open the emulator tab
  4. select 4.3 from drop down an launch
  5. once the emulator is active, go to to IOS simulator menu > hardware > device > select retina screen
  6. hit stop button in TiDEv, the relaunch emulator

results: a non retina version is launched.

As per Blain, we should do a quit not Kill on stop to preserve settings
