Description

From what I've been able to research, it is not possible on the iPhone to change the image for pins/markers that are displayed on a map created with Titanium. The only way to use custom markers on a map if you are using Titanium is to use Google Maps instead of the native Titanium.Map. This is a very big issue and would hope this can be addressed in a future release.

In summary, the request is to add the ability to use custom images as map pins/markers for the iPhone/iPad. So this would be a custom image for the Titanium.Map.Annotation.
