Description

Feature

On iOS, we have the Titanium.Media.VideoPlayer.thumbnailImageAtTime function which takes a time and an options array, and returns a blob thumbnail. Android doesn't have this function implemented. So we can either add it to Android, or update the documentation saying it is not available for Android.
