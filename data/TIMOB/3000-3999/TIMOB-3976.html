Description

Description: While testing using KitchenSink, specifically Base UI>Views>WebView Controls, I navigated away from the Google page loaded at the start of the test. Clicking the "reload" button loaded the Google page again instead of reloading the page I was on. Steps to reproduce: 1) Install KitchenSink 1.7.0 on the Verizon iPhone 4 2) Choose Base UI>Views>WebView Controls 3) When the page has loaded, choose any link on the page 4) Click the "reload" button at the bottom of the page Result: The main Google page loads and displays Expected Result: The page you navigated to should reload and display Notes: Does not occur with KS 1.6.2
