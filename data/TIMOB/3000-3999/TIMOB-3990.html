Description

Description: While testing KitchenSink, Controls>label>Basic, clicking the "change label 2" button once, then again will change the text as expected, but the text goes into an intermediate state before being enlarged. Steps to reproduce: 1) Install KitchenSink 1.7.0 on a supported iOS device 2) Launch Controls>label>Basic 3) Click "change label 2" button 4) Click "change label 2" again to restore the label to it's original format Result: Before the label is restored to the large-font, shadowed text, it flashes into some other format prior to its end state. Expected Result: Smooth text transformation between two formats Notes: Also occurs with KS 1.6.2
