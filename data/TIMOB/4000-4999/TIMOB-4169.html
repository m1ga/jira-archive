Description

Description: With a few steps, it's possible to get the appcelerator media controls to appear partially behind the tab bar. Steps to reproduce: 1) Install KitchenSink on Verizon iPhone 2) Navigate to Phone>Play Movie>Local 3) play the movie. 4) enter fullscreen playback 5) Rotate to landscape 6) rotate to portrait 7) Return from fullscreen playback Result: Media controls partially behind tab bar Expected Result: Media controls fully visible Notes: Could not reproduce on 1.6.2 because test does not allow orientation change
