Description

Description: If you set the Duration between frames in Base UI> Views> Image Views> Animate to 0, the app unexpectedly quits. Steps to reproduce: 1) Using KS 1.7.0 or KS 1.6.2, install on an Android device 2) Navigate to Base UI> Views> Image Views> Animate 3) Set duration to 0 ms 4) stop animation (to take the ms change) 5) Start animation Result: Dialog stating app unexpectedly quit Expected Result: Super fast animation Notes: Occurred on Droid, Nexus S, KS 1.7.0 and KS 1.6.2
