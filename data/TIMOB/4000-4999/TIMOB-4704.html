Description

Sample failing drillbit code: 
var doc = Ti.XML.parseString(this.testSource["nodes.xml"]);
var node = doc.getElementsByTagName("node").item(0);
var attr = doc.createAttribute("newattr");
node.setAttributeNode(attr); // NPE

Comments

  1. Bill Dawson 2011-07-25 Problem is that, as per the [spec](http://www.w3.org/TR/DOM-Level-2-Core/core.html), setAttributeNode returns null if an attribute with same name doesn't already exist. If it does already exist, it returns the attribute being replaced.
