Description

There is currently an issue (TISTUD-559) with XML serialization outputting empty XML tags when they have no contents, as opposed to a start tag/end tag pair. Some of our build scripts rely on end-tag searching (GUID placement for command-line builds and modules) and so we need to modify them so that they can handle empty tags.
