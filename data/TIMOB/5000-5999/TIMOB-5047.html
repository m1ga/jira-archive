Description

in iOS5 the barImage for a windows navbar is no longer working. This comment on StackOverflow explains the 2 ways it can be fixed, or can at least be a jumping off point - http://stackoverflow.com/questions/5575821/custom-nav-bar-styling-ios/6389991#6389991 

var tabGroup = Titanium.UI.createTabGroup();

var win1 = Titanium.UI.createWindow({  
    title:'Tab 1',
    backgroundColor:'#fff',
    barImage:'barImage.png'
});
var tab1 = Titanium.UI.createTab({  
    title:'Tab 1',
    window:win1
});

tabGroup.addTab(tab1);  

tabGroup.open();
