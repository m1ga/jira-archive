Description

Mobile apps can currently get (forward and reverse) geolocation lookups from Titanium by way of a call to our API. We are deprecating support for this service (see: CLOUDENG-376). Titanium Mobile should directly access a service, such as [Google's Geolocation API](http://code.google.com/apis/maps/documentation/geocoding/) directly. For example, version 3 of the Google API doesn't require API keys and rate limits by IP/device.
