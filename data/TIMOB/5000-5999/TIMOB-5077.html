Description

Problem:

In the KitchenSink Hide Soft Keyboard example, pressing the home button leaves the soft keyboard displayed.

Reproducible steps:

1 Open KitchenSink Basic UI -> Hide Soft Keyboard (Android) 2 Press down arrow to focus on Request Hide Keyboard button. 3 Press up arrow to focus on TextField. Soft keyboard shows. 4 Press Home button. The app closes, but the soft keyboard stays displayed instead of closing.
