Description

Description: While testing Kitchensink, Base UI>Views>Image Views>Image toolbar, I rotated the device with the toolbar and soft KB up and the toolbar was partially behind the KB. See screenshots. This condition persists once it occurs through additional rotations. Steps to reproduce: 1) Install KS 1.8.0 on iPod 3rd gen. 2) Navigate to Base UI>Views>Image Views>Image toolbar 3) Tap label "just focus here for keyboard" 4) rotate device Result: Toolbar rendered behind soft KB Expected result: Toolbar fully visible Notes: Occurs on 1.7.2 as well.
