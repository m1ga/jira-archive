Description

Description: After running the tests at Phone>Orientation>, the Nav bar in the next table view is rendered smaller than required. On iPod, there are two navbar sizes, one for landscape, one for portrait. It looks like the application in in such a state that the smaller navbar is rendered. This is rectified by rotating the device. Steps to reproduce: 1) Install KS 1.8.0 on iPod 3rd gen. 2) Navigate to Phone>Orientation 3) Tap "set portrait" button 4) go back 5) Tap "Contacts" to view test list 6) Examine the nav bar Result: The nav bar is smaller than the normal portrait navbar, making the top row in the tableview look large Expected result: Portrait-sized navbar Notes: Occurs with 1.7.2 as well
