Description

When a button with no specific style is created, it always has an "extra border" on the bottom. This is visible on iPad. Sample Code {noformat} Titanium.UI.setBackgroundColor('black'); var window = Titanium.UI.createWindow({ title:'Hello', backgroundColor:'black' }); var button = Ti.UI.createButton({ width:150, height:50, title:'My Button', top:200, left:100 }); window.add(button); window.open(); {noformat}
