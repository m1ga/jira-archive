Description

Problem

When you name a project in Titanium with unicode characters like swedish characters åäö you got errors when you run the application. It happens in Android, and iOS.

Reproducible steps:

1.Create a project in Titanium Studio with the name åäö 2.Name the App Id with a different name than åäö 3.Run the project.

Expected behavior

User wants to have support with unicode characters, and name his applications with swedish characters like åäö.

Comments

  1. Chris Barber 2016-08-22 This was fixed a long time ago and I cannot reproduce using Titanium SDK master (currently 6.1.0). Not sure when exactly it was fixed, but you can create a project and use unicode characters in the name of project. The Titanium build system can handle the special characters in both file paths and in the contents of files. Both the iOS Simulator and Android emulator support them too.
