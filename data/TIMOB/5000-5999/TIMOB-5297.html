Description

When building an app that starts in landscape mode there is a orientation change animation just after the launch image disappears and the first window is opened. This is not the case with landscape only app developed with xcode: the launch image is displayed and the first window loads in landscape mode correctly without the unnecessary animation. Se app.js and tiapp.xml below. Note that it doesn't matter if you put portrait mode in tiapp.xml instead. 
var win = Titanium.UI.createWindow({  
    backgroundColor:'#fff',
    orientationModes:[Ti.UI.LANDSCAPE_RIGHT]
});
win.open();

//iphone section only
<iphone>
    <orientations device="iphone">
    <orientation>Ti.UI.LANDSCAPE_RIGHT</orientation>
</orientations>
Edit: This only applies if you have true in tiapp.xml. If you set this to false there is no animation, but then there will be a status bar on your splash which is not desired.
