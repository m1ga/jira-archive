Description

Description: While testing KS 172 on iOS 5, Base UI>Views>Table Views>Table View (layout 4), I noticed a translucent bar at the bottom of the table view. This bar acts like a footer as it remains in place until you scroll the table to the bottom at which point it detaches from the bottom of the view Steps to Reproduce: 1) Install KS with SDK 1.7.2 onto iOS 5 device 2) Navigate to Base UI>Views>Table Views>Table View (layout 4) 3) Scroll the table to the bottom Results: The "footer" is visible and remains in place until the table is scrolled completely to the bottom. Expected results: No errant footer. Note: Does not occur with iOS 4.3x Occurs with SDK 1.8.0.v20110922105247 Please see attached image for details
