Description

Description: When using KS 1.7.2 (synced from 1_7_X branch) and built with 1.7.2 SDK on an iPad 2 running iOS 5.0, launching the test and clicking the button "Close/Animate Tab Group" results in a white screen with only a status bar. Running the same app on an iPhone running iOS 5.0 results in the tab group closing as expected. Steps to reproduce: 1) Sync to 1_7_X branch 2) Import KS into Studio 3) Target iOS 4.3 or iOS 5.0 and SDK 1.7.2 GA 4) Install on iPad 5) Navigate to Base UI>Tab Groups and click "Close/Animate Tab Group" Result: Screen fades to white screen with only status bar Expected: Tab group close and returns to base UI tab Notes: Occurred on 1.8.0 KS and 1.7.2 KS Occurred on 1.7.2 SDK and 1.8.0 SDK
