Description

Description: While testing Base UI>Views>Web Views>Webview controls, I clicked on the images link then clicked "back" and then "reload". Instead of reloading the main google page, the images page was opened and reloaded. This does not occur with iOS 4.3.x Steps to reproduce: 1) Install and launch KS on iOS5 device 2) Navigate to Base UI>Views>Web Views>Webview controls 3) Click a link 4) Click the "back" button to return to Google homepage 5) Click the "reload" button Result: The link you clicked in step 3 will open and reload. Expected: The main google page reloads. Note: This seems independent of SDK. A device with iOS 5 but with SDK 1.7.2 also exhibits the behavior.
