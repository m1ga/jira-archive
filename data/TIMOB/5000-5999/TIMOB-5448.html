Description

Description: While testing KS, Base UI>Modal Windows, I was able to get the UI out of sync with the device. This initially resulted in a smaller "landscape-style" navbar, but an additional device rotation had the status bar rotate alone. See images for details. Steps to reproduce: 1) Install KS on device 2) Navigate to Base UI>Modal Windows 3) set picker to "Flip horizontal" 4) Click "show modal" 5) Rotate to landscape 6) Close the modal window 7) Rotate again Result: On modal close, the navbar will be small with some whitespace below. When rotated as in step 7, the status bar will rotate, but the rest of the UI will not. Expected: UI remains in sync with device Note: Did not occur with iOS 4.3.x or 1.7.2 SDK
