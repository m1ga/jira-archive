Description

Description: While testing KS, Mashups>Facebook>login/Logout, I started filling out the login account name and then rotated the device with the keyboard up. All the UI with the exception of the login window rotated with the device. See image for details. Steps to reproduce: 1) Install KS on device 2) Navigate to Mashups>Facebook>login/Logout 3) Set force Dialog to "true" 4) Click "login" 5) Focus on one of the text fields to get the Soft KB 6) Rotate the device Result: The UI rotates, but the login window does not. Expected: The login window should rotate with the rest of the UI.
