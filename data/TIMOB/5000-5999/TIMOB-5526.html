Description

when the hide button was pressed twice various kitchen sinks, the hidden object could not be shown. We implemented workarounds, but this should be fixed in the SDK. See ui_methods, textfield, textarea here is the workaround we implemented (in textarea) showHide.addEventListener('click', function(){ if (ta1.isvisible){ ta1.hide(); ta1.isvisible = false; }else{ ta1.show(); ta1.isvisible = true; } })
