Description

The leftNavButton/rightNavButton is missing shadow when using a custom image. The shadow is visible when using a "systemButton" icon. !Screen Shot 2011-10-29 at 20.11.24.png! Testcase: Just place the "switch-language.png" and "switch-language@2x.png" in the "Resources/iphone" folder. app.js: 
var switchLanguage = Ti.UI.createButton({
	image: "iphone/switch-language.png"
});

var actionButton = Ti.UI.createButton({
	systemButton: Ti.UI.iPhone.SystemButton.ACTION
});

var win = Ti.UI.createWindow({
	leftNavButton: switchLanguage,
	rightNavButton: actionButton
});

win.open({
	modal: true
});
