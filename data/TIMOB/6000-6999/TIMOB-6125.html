Description

Description: The UI elements in KS Base UI > Views > scroll views > scrolling tabs are drawn so slowly that you can perceive the draw start location and the UI element looks painted in from one side. On the LG Slate, there is a couple second delay before the UI elements are rendered, and they may appear piecemeal, or all together with no perceptible drawing process. Steps to reproduce: 1) Install KS on device 2) Navigate to Base UI > Views > scroll views > scrolling tabs 3) Watch as the UI elements render Result: Delay in some cases, slow draw in all others. Expected: Quick rendering of UI elements with no delay in render start
