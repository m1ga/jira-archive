Description

Issue

This sample app, when switching back and forth(only hiding and showing views) will eventually release those views and just show a blank window. This happens on both iOS5 and iOS4.3. In further debugging we found that if you remove the text, the views never get release and function as expected.

Testing Steps

1. Install app on iOS Device(iOS4 or 5) from the attaches app.js file 2. Open App and click back and forth between the 'News' button and the 'Blogs' Button 3. App will eventually release the container view so the page is blank

Additional Info

Customer has been struggling with this for a while and ony has a problem in iOS5. The only reason iOS4 is brought up here is due to the issue discovered while trying to debug the iSO5 problem.
