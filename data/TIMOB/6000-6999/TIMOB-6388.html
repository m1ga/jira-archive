Description

Two extra empty browser windows are launched in Firefox on first run or page refresh. The URLs are: https://api.appcelerator.net/p/v2/mobile-track?callback=html5_jsonp695648 https://api.appcelerator.net/p/v2/mobile-track?callback=html5_jsonp864800 Steps to Reproduce: 1. Open Studio, create default MobileWeb app. 2. Launch with Firefox (browser can be set in the Run Configurations menu). Expected Result: One new window should load in Firefox showing the content of the app. Actual Result: Three new windows open (the app and two extra URLs). The extra URLs also launch on page reload. Note: This behavior is consistent between operating systems. Only found behavior in Firefox. Screenshot and logs attached.
