Description

Description: In KS, on the Base UI>Animation>transitions test, Rotating the device while the page_curl animation is rendering causes the UI to not use the full extent of the display when the animated window is closed. The timing of the device rotation is critical in reproducing this bug. Steps to reproduce: 1) Install KS built with SDK 1.8.0.1 2) Navigate to Base Base UI>Animation>transitions 3) Click the "full window" button and immediately rotate the device 4) Click the close button Result: The UI now only uses about two thirds of the display area. Expected: No changes in UI. Notes: Also occurs with KS and SDK version 1.7.5
