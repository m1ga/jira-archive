Description

In Rhino, debugger integer variables display as floats. Integers display correctly in V8 and iOS. This is a general issue that could be tested in any Titanium Mobile project (containing integers and breakpoints) running Rhino in debug mode. Tested in: /titanium_mobile_tests_internal/FeatureTest/Debugger Steps to Reproduce: 1. Create a new project in studio called debug_test . 2. Move the .js files from the debug test project in titanium_mobile_tests_internal.  3. Refer to the test plan at http://wiki.appcelerator.org/display/tp/Debugger+Test+Plan . 4. Execute test case 1014 up to Step 5 (in Rhino). Expected Result: Variables - Name: y, Value: -1 Actual Result: Variables - Name: y, Value: -1.0
