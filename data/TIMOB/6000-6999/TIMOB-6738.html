Comments

  1. Chris Barber 2016-08-22 titanium.py has been deprecated in favor of the Titanium CLI. The Titanium CLI was introduced alongside Titanium SDK 3.0.0 and supports a -v and \-\-version options to display the CLI version as well as a ti sdk command to display Titanium SDK information. Needless to say, we will not be adding version information to titanium.py.
  2. Lee Morris 2017-03-22 Closing ticket as the issue will not fix and with reference to the above comments.
