Description

A feature request has come in for the ability to style text on annotations. This would need to be a completely new annotation type which displays an object styled as a label. Will probably have to accept Ti.UI.Label objects for title and subtitle properties to ensure that they're styled correctly, and that we support every styling feature that a label does. Android will need a parity ticket if this feature is implemented.
