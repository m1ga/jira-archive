Description

for Ti.Map.Annotation you should be able to set the image property to a blob in order to support dynamic annotation creation. Right now it only accepts a URL to an image file. Comment from customer: We're facing a technical challenge that can actually make or break us at this point. We have a deadline at the end of january to build a mobile app in titanium that can closely mimic our current app for iOS. Our problem: We need the ability to add labels with dynamic text to annotations. We would actually love to be able to add views to the annotation view but labels would be enough for our current needs.
