Description

Ti.include blocks global variables incorrectly.

Repro Steps:

Create a project to include the code below. 
//app.js
var win = Ti.UI.createWindow();

var headerView = Ti.UI.createView();

Ti.include('test2.js');

win.add(headerView);
win.open();

//test2.js
var otherView = Ti.UI.createView();

headerView.add(otherView);
