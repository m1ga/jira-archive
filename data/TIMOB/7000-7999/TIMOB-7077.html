Description

Build a "mobile web inspector" that would popup in a window on load and allow developers to show the Ti element tree, inspect individual Ti elements, events, view console output, loaded modules, cache status, Ti.Database browser, Ti.Filesystem browser, Ti.Network traffic, resize window, and search. This inspector would only be visible when in "development" mode. For desktop browsers, the inspector would be a separate browser window. For devices, we may choose to not support device debugging, create a bridge to a desktop browser, or use a dialog (which would obscure the view).
