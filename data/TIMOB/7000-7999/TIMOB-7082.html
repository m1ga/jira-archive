Description

Detect if the browser is capable of running Titanium and if not, display a message warning the user. This screen should be easily customizable by the developer. The browser detection should happen after the loader but before the project's app.js is downloaded. Create a default warning template with no dependencies (i.e. inline images, styles, etc) in the mobileweb/src directory. At compile time, if the Resources folder does NOT contain an overriding template, then use the default warning.
