Description

Problem

Neither window open or focus events are fired. Note that both work on Android, albeit with bugs (that can be worked around). See linked ticket, and its linked tickets.

Test case

var win = Ti.UI.createWindow({
  backgroundColor:'red'
});
win.open();

win.addEventListener('open', function() {
Ti.API.info('------- Window open event fired ------');
});

win.addEventListener('focus', function() {
Ti.API.info('------- Window focus event fired ------');
});

Logs

Nothing output to the logs.

Comments

  1. Paul Dowsett 2012-01-16 open and focus eventlisteners should go before the open method. I will raise an apidoc ticket to convey this advice. Hence, closing as invalid
  2. Lee Morris 2017-03-09 Closing ticket as invalid.
