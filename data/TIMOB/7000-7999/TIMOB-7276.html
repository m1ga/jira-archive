Description

Details: After installing Titanium Apps specifically to iPod 3rd Gen (4.02), default and custom icons will intermittently disappear on the device. See attachment. To get the icons back, you will need to reinstall the app again onto the device. Restarting the device will not bring the icons back. This might be related to an Apple bug: https://discussions.apple.com/thread/2753249?start=0&tstart=0 Steps to reproduce: 1. Create a Titanium project using default or custom icons 2. Install project to iOS devices. Preferably an iPod Actual: Icons will disappear intermittently. Expected: Icons should not disappear. Note: Filing this bug in case other Titanium users come across this issue.
