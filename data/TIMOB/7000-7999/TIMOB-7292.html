Comments

  1. Michael Pettiford 2012-01-20 This also occurs with Nook Color with the same environment as in the ticket
  2. Marshall Culpepper 2012-01-24 KitchenSink uses our Maps API which requires the Google Maps API be installed on the device you deploy to. The Kindle Fire (and Nook) don't have the Google Maps library installed, and the error reported is the standard error that occurs when you try to install an app with a Maps dependency on a device without Maps. In the meantime, KS-Nook should work on the Kindle Fire.
