Description

Steps to reproduce: 1. Launch KS from 1_8_X branch 2. Go to Base UI > Tab groups 3. Press "Add/Remove Tab" Actual: "Mashups" tab does not dynamically change to "More" Expected: "Mashups" tab should dynamically change to "More" 4. Press "Close/Animate Tab Group" Actual: "Mashups" tab will change to "More" after tab group closes Expected: Should not see any changes to tab after pressing this button Note: This new behavior does not occur in 1.8.0.1 KS
