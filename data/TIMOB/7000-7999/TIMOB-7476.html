Description

Problem:

Titanium.UI.getCurrentWindow does not exist in 1.8.0.1.

Reproducible steps:

1. Run the code below with 1.8.0.1 Titanium SDK. Please see the 1.8.0.1 Titanium SDK-Android API 2.3.3-Bad Behavior.png attached image. Notice that the method getCurrentWindow does not exist.

Expected behavior:

Customer wants to use the method getCurrentWindow.

Tested with the next specs:

Titanium Studio, build: 1.0.9.201201262154 Simulator Android API: Android 2.3.3 Titanium SDK: 1.7.5 & 1.8.0.1 Samsung Fit Android version of the device:2.2.1

More info:

http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.UI.getCurrentWindow-method.html
