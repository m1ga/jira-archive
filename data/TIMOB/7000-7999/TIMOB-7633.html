Description

Steps to reproduce: 1. Create a default Titanium Mobile project 2. Replace the resources directory with the one attached 3. Open Xcode > Window > Organizer and click on Console for your device 4. Run app and press the "table view touchend" row 5. Press the close window button 6. Press the table view search button Actual Results: After step 4: Two touch end events are fired After step 6: Four touch events are fired Expected Results: A single touch end event is fired *Note this behavior occurs in 1.8.1, 1.8.0.1, and 1.7.5*
