Description

Problem

Project must be cleaned between subsequent runs against the simulator to remove build error

Expectations

Expect that when project is run against the simulator, that I don't have to clean the project each time to remove the symlinks to allow build to continue

Observations

* This particular project is the only one in which we copy files from /Resources (Resources/library/html) out into the /Library (/Library/Private Documents/) in order to update/overwrite them. * Switching between sdks resolves problem with first run. * Subsequent runs against the same sdk has problem * Manually deleting files from simulator directory does not resolve problem. * Removing app from simulator does not resolve problem.
