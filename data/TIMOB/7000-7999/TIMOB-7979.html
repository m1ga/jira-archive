Description

Currently we don't support the Contacts twitter field, which was introduced in iOS5. See [TiContactsPerson.m](https://github.com/appcelerator/titanium_mobile/blob/master/iphone/Classes/TiContactsPerson.m). This is to request for it to be added to [Titanium.Contacts.Person](http://developer.appcelerator.com/apidoc/mobile/latest/Titanium.Contacts.Person-object) for iOS, and for Android if possible. stramer gave the go-ahead for this request to be raised.
