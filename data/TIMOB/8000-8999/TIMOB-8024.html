Description

TextArea return event fails to fire. Works on iOS. Works in 1.8.2. Regression. Steps to Reproduce: 1. Run code, press 'return' on keyboard (focus keyboard if necessary). 
var win = Ti.UI.createWindow({
	backgroundColor:'blue'
});

var textArea = Ti.UI.createTextArea({
	value:'TextArea',
	top:'20%',
	height:'30%',
	left:'10%',
	right:'10%'
});

textArea.addEventListener('return',function(){
	alert('return event fired');
});

win.add(textArea);

win.open();
Expected Result: Return event should fire, alert box appears. Actual Result: No event fired.
