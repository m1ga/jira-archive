Description

Problem

Setting top=null on the view causes bottom to not be recognized

Steps to Reproduce

Run the sample code below: {noformat} var win=Ti.UI.createWindow({ backgroundColor : 'blue' }); var view=Ti.UI.createView({ width: 500, height: 300, top: null, bottom: 0, left: 0, backgroundColor: 'green' }); win.add(view); win.open(); {noformat} On 1.8.2 the "green" view fills up the entire window(screenshot1) but on the 2.0 CI, with top=null and bottom as 0, the view still is at the top instead of anchoring in the bottom (screenshot2)
