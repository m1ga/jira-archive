Description

On KS>Controls>TextField>The Rest - clear button is not displayed on the input field when 'Clear Button','Clear Button(A)','Clear Button(F)','Clear Button(B)'button is pressed. Steps to Reproduce: 1:enter text in input field 1:tap on 'Clear Button' 2:tap on 'Clear Button(A) 3:tap on 'Clear Button(N),tap on the input field to focus. 4:tap on 'Clear Button(F) 5:tap on 'Clear Button(B) Actual Result : clear button don't get displayed on the input field when required Expected Result: tap on 'Clear Button' Should make visible the button on the input field. tap on 'Clear Button(A)' should make the button invisible on the input field. tap on 'Clear Button(N)' should make visible the button on the input field, on focus tap on 'Clear Button(F)' should make button visible on the input field,on Blur tap on 'Clear Button(B)' should make button visible on the input field
