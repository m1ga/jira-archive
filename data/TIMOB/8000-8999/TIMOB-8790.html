Description

Setting up the SeparatorColor property as null displays separator color as yellow. Steps to Reproduce: 1. Open up the kitchen Sink application. 2. Tap on base UI 3. Tap on Views 4. Tap on table Views. 5. Tap on Set Row data 6. Tap on Set2 Actual result: After step 6: Could see 'Set TableView 1','Set TableView 2'Set TableView 3' and three rows of Row row separator was single line and yellow color for the separator color Expected Result: After Step 6: Should see 'Set TableView 1','Set TableView 2'Set TableView 3' and three rows of Row row separator should be single line and no color for the separator color Note: This behavior is observed 1.8.2 , 2.0.1 and 2.1.0.
