Description

Description: While testing "Options Dialog" on iPhone and iPod, I noticed, that options dialog does not change orientation after device is rotated from portrait to landscape. This bug exists on 1.8.2 and 2.0.1. Does not reproduce with iPad devices. Steps: 1. Run KitchenSink 2. Go to Base UI-Views-Options Dialog 3. Press "Show Dialog 1" 4. Rotate device to landscape mode Expected result: Options dialog should change orientation fron portrait to landscape Actual result: Options dialog does not change orientation
