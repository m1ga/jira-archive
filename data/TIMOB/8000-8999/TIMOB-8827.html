Description

The socket support in platform currently lacks support for UDP and multicast. Dawson has created a UDP module that we should investigate rolling into platform. Any inclusion into platform should be done after evaluating how UDP *should* work in relation to our existing TCP socket code and API structure. Once we know how UDP should look API wise, evaluate the UDP module to see what level of changes are required to make the module suitable for inclusion into platform.
