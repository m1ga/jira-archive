Description

Problem

Using the backgroundGradient property when creating a label object the following error is generated: [ERROR] Adding an event listener to a proxy that isn't already in the context. This is tested on 2.0.1GA2 as well as the 1.8.2 Release.

Sample Code

{noformat} var win = Ti.UI.createWindow({ backgroundColor:'white' }); win.open(); var signonview = Ti.UI.createView({ top:10, left:10, right:10, bottom:10, backgroundColor:'transparent' }); var label = Ti.UI.createLabel({ text:'alreadyNetworkSubscriber', backgroundGradient:{ type:'linear', colors:['#929292','#717171'], startPoint:{x:0,y:0}, endPoint:{x:0,y:45}, backFillStart:true }, shadowColor:'#282828', shadowOffset:{x:0,y:1}, top:0, height:45, width:290, textAlign:'center', color:'#fff' }); signonview.add(label); win.add(signonview); {noformat}
