  3. Neeraj Gupta 2012-05-07 The feature called 'Enable access for assistive devices' is found in the Universal Access preference pane. It needs to be enabled in order for any GUI AppleScripts to run. You can enable it from Terminal as well using the following command. To turn it on, type this in Terminal: sudo touch /private/var/db/.AccessibilityAPIEnabled To then disable it, type this: sudo rm /private/var/db/.AccessibilityAPIEnabled
