Description

Problem

The Slider is not working properly if max property is different from 100. Setting a value to the slider has no effect.

Test case

var statusSlider = Ti.UI.createSlider({
	min : 0,
	max : 150,
	minRange : 0,
	maxRange : 150,
	enabled : true,
	width : '180dp',
});

statusSlider.setValue(75);
