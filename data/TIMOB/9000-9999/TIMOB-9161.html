Description

Near as I can tell, both Android and iOS expose the Blob.type property, but not the constants for the type values (TYPE_IMAGE, TYPE_DATA, etc.) At present, type is not documented in Blob.yml because it's not terribly useful without the constants. I think we should either expose the constants or not expose the 'type' property.
