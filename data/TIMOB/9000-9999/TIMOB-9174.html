Description

Problem Description

Customer is working on creating a mobile web version of their app. The app runs fine in Safari @ iOS, but it's presenting a bad performance in BBOS7 native browser, even under the same conditions (running over the same wifi).

Expected Results

The same performance between different OSes

Actual Results

Bad performance in BBOS7

Comments

  1. Chris Barber 2012-05-18 The 2.0.1 and 2.0.2 CI build does not include support for BBOS7 devices. Furthermore, it also lacks some serious performance improvements we've done. We highly recommend you visit our CI build download site and download the latest master CI build: http://builds.appcelerator.com.s3.amazonaws.com/index.html. Once you install the latest master CI build, try your application again and if the performance is tolerable, let's close this issue. We have several other issues to improve performance across all environments.
  2. Chris Barber 2012-06-04 The bad performance is partly due to the lack of power of the BlackBerry devices. Mobile Safari for iOS is much faster making it impossible for BlackBerry's web browser to run just as fast. Upon further investigation of the example link, the performance is greatly impacted by the poor design of the application and the lack of precaching includes and requires.
