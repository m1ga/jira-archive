Description

The barcode module 2.3.0 in Titanium crashes when an i18n/en/strings.xml file is pressent in root of an application.

Repo Steps

1. Create a new Mobile App select the Single Window Template (it will create the i18n folder). Copy the content of the Example app.js from the module folder in to your project's app.js 2. Run the app in Android device. Click the Scan Code button and the app will crash.
