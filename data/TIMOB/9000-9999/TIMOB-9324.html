Description

When Ti.UI.Switch control is added to some view it fires 'change' even first time. Expected behavior is that it fires 'change' event only when user clicks on it. Tested with Titanium SDK 2.1.0 on iOS 5.0 Simulator 

var win = Ti.UI.createWindow();

var simpleSwitch = Ti.UI.createSwitch({
	value : false
})

win.add(simpleSwitch);

simpleSwitch.addEventListener('change', function(e) {
	alert('Switch value: ' + e.value)
})

win.open();
