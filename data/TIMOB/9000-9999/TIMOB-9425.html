Description

Description: Clicking a button "Toggle Opacity" doesn't change opacity if background color or background image was changed. Only second click of "Toggle opacity" changes color intensiveness. Not a regression, the behavior exists on 2.0.2 as well. Steps to reproduce: 1. Run KitchenSink 2. Go to Base UI - Window Properties 3. Press "Toggle opacity" 4. Press "Change BG Color" or "Change BG Image" 5. Press "Toggle opacity" one more time after BG Color (or BG Image) was changed Actual result: Opacity hasn't changed Expected result: Opacity should be changed
