Description

Image disappears when its Zooming factor value is set to zero or less This is not a regression.It even happens on 2.0.2 Steps to reproduce: 1. Run the below sample code on device. 2. Pinch and Zoom the Picture Sample code 1 
var _window=Ti.UI.createWindow ({
		backgroundColor : 'black'
		})	
		var photosView = Ti.UI.createScrollableView({
		    backgroundColor:'black',
		    width : 320,
		    height : 450,
		    showPagingControl:true,
		    pagingControlHeight:20,
		});
		
		var imgArray = [];
		var img1 = Ti.UI.createImageView({
		    image:'http://media.koreus.com/200606/1image-illusion-optique14-mini.jpg'
		});
		var imgWrapper = Ti.UI.createScrollView({
			maxZoomScale:0.0,
		
		});
		imgWrapper.add(img1);
		
		imgArray.push(imgWrapper);
		photosView.views = imgArray;
		photosView.views = imgArray;
		_window.add(photosView);
		
	_window.open();
Expected Result: Should be able to zoom and pinch the picture without any errors Actual Result: Image disappears.
