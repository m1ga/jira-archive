Description

Description: The keyboard cannot be dismissed by pressing "Done" button. Landscape mode: there is no "hide keyboard" is visible on a screen."Done" buttons: one is in a text field, another is on a soft keyboard. Press "Done" on a keyboard disables an input, repeatable press on "Done" flickers the screen. The rotation to the portrait mode stretches a keyboard horizontally and shrinks it vertically, cutting of a right part of a keyboard. Not a regression, reproducible in 2.0.2 Steps to reproduce: 1. Run KitchenSink 2. Go to Base UI - Hide Soft Keyboard(Android) 3. Tap in a text field. 4. Rotait the device. 5. Press "Done" or "hide keyboard", if visible. Expected result: Soft keyboard should disappear. Actual result: See screen shots.
