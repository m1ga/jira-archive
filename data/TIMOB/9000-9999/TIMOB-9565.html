Description

While testing KitchenSink, the text field hides behind the soft keyboard for the KitchenSink>Window Soft Input (Android) test case. This is not a regression.It even occurs on 2.0.2 Steps to Reproduce: 1. Install kitchensink onto the device and launch it. 2. Tap on Base UI tab. 3. Click on Window Soft Input (Android) 4. Tap on SOFT_INPUT_ADJUST_PAN, and then tap on Launch 5. Tap on text field, after input , tap next on soft keyboard Actual Result: After Step 5,The text field is hidden behind the soft keyboard. Expected Result: After Step 5,should be able to enter text in the text field,and tap on 'next' should shift the focus to next text field.And,the text field should not be hidden.
