Description

Reload button redirects to the previous page instead of reloading the current page. This is not regression. This occurs since 1.8.2 Steps to reproduce: 1. Go to Kitchensink> BaseUI> Views> WebViews> WebView Controls. Google home page is loaded. 2. Type something in text area and search. User is redirected to second page. 3. Press back. User is redirected to Google home page. 4. Press fwd. User is redirected to second page. 5. Press reload. Expected: Current page i.e. second page should be reloaded. Actual: User is redirected to Google home page.
