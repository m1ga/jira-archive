Description

Description: While testing KS Contacts I noticed that the typed in text field's content disappears after "Done" button pressed. This is not a regression, occurs on 2.0.2 as well. Steps to reproduce: 1. Run KS - Phone - Contacts - Search by ID. 2. Tap on a text field. 3. Type ID. 4. Press "Done" button on a soft keyboard. Actual result: The typed in content disappeared. Expected result: The text field's content should not disappear.
