Description

Description: During KS testing I noticed, that if "Partial Curl" chosen, Modal Window does not change orientation from landscape to portrait mode. There is also no rotation to the portrait mode after "Close" button pressed and "Main" Window appears. Not a regression, occurs in 2.0.2 as well. Steps to reproduce: 1. Run KitchenSink in landscape mode 2. Go to Base UI - Modal Windows 3. Scroll down to "Partial Curl" in first column 4. Press "Show modal" 5. Rotate device to portrait 6. Press "Close" button Expected result: Modal and "Main" windows should change orientation Actual result: Modal and "Main" windows don't change orientation
