Description

The webview with background image shows a white blank space when zoom out on webview is done. Not a Regression. The issues exists as far as 1.8.2 The issue occurs on Android 2.X and 3.X. It does not happen on Android 4.X. Steps To Reproduce: 1. Open Kitchen Sink application 2. Go to Base UI -> Views -> Web Views -> Local URL 3. Swipe on the screen to get the zoom control 4. Click on the Zoom out button one or more times Actual: A white blank space start to show below the web view Expected: Zoom Out should not show any white space
