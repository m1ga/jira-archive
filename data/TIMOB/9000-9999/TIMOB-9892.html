Description

The Android Emulator is improperly displayed and the touch events are inaccurate on the MacBook Pro Retina Display. This is an android issue: http://code.google.com/p/android/issues/detail?id=33383 Attached image courtesy of: http://stackoverflow.com/users/147333/corey-d Steps to Reproduce: 1. Buy a MacBook Pro w/Retina Display. 2. Install Titanium Studio and the Android SDK. 3. Open Titanium Studio and create a new Titanium Mobile project. 4. Run project in Android Emulator. Actual Result: Emulator is poorly displayed. Expected Result: Emulator should display properly and receive events in the correct location.
